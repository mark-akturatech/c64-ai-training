BASIC Program Area ($0800-$9FFF)
---------------------------------

$0800   Link Area               Must contain 0 to allow RUN
$0801-$9FFF  BASIC Program      Program storage area (38,911 bytes)
$8000-$9FFF  Cartridge ROM      Optional external ROM (8192 bytes)

Cartridge Structure:
  $8000-$8001  Cold reset entry point
  $8002-$8003  Non-maskable interrupt handler
  $8004-$8008  Signature string "CBM80"


BASIC ROM ($A000-$BFFF)
------------------------

$A000-$BFFF  BASIC ROM/RAM      BASIC interpreter or RAM (8192 bytes)

Visibility controlled by processor port $0001 bits #0-#2.

Key Notes
=========

Bank Switching:
  The processor port at $0001 controls visibility of BASIC ROM ($A000-$BFFF),
  KERNAL ROM ($E000-$FFFF), Character ROM ($D000-$D7FF as text), and I/O area
  ($D000-$DFFF).

VIC Bank Selection:
  CIA#2 port A bits 0-1 select which 16KB bank the VIC-II chip accesses,
  allowing video memory location flexibility.

Default Configuration:
  The standard setup places BASIC programs at $0801, screen memory at $0400,
  and color RAM at $D800, with BASIC and KERNAL ROMs visible and I/O active.
