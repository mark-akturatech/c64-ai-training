# 6581 SID — Absolute Maximum Ratings

**Summary:** Absolute maximum ratings for the 6581 SID: VDD, VCC, analog input Vina, digital input Vind, operating temperature Ta, and storage temperature Tstg (values and units preserved).

## Absolute Maximum Ratings
The table below lists the absolute maximum electrical and thermal limits for the 6581 SID. These are stress limits; operation beyond them may cause permanent damage (do not use as normal operating values).

- VDD (Supply Voltage): -0.3 to +17 VDC
- VCC (Supply Voltage): -0.3 to +7 VDC
- Vina (Analog input voltage): -0.3 to +17 VDC
- Vind (Digital input voltage): -0.3 to +7 VDC
- Ta (Operating temperature): 0 to +70 °C
- Tstg (Storage temperature): -55 to +150 °C
