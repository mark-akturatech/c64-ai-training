# PETSCII / C64 Screen Codes $20-$3F (Dec 32–63)

**Summary:** Lists the PETSCII / C64 screen codes $20-$3F (decimal 32–63) mapping hex values to printable characters: space, punctuation, digits 0–9, and symbols : ; < = > ?. Useful for screen RAM, text output, and character mapping on the VIC-II.

## Screen codes $20-$3F
This chunk enumerates the C64/PETSCII printable screen codes in the hex range $20–$3F (decimal 32–63). Each byte value corresponds to the character shown on the C64 character ROM when used in screen memory or text output. The range includes space, common punctuation, the ASCII-like digits 0–9 ($30–$39), and the symbols colon through question mark ($3A–$3F).
