# COMMODORE 64 — Manual Contents (Programmer's Reference Guide)

**Summary:** Table of contents and scope for the COMMODORE 64 Programmer's Reference Guide covering Commodore BASIC dictionary, machine language overview, KERNAL routines, I/O programming (disk, cassette, printer, modem, joysticks, lightpens), graphics (sprites, programmable characters, bitmaps), SID sound/music, and optional languages (CP/M and high-level languages).

## What's Included
- BASIC dictionary
  - Complete Commodore BASIC words, commands, statements and functions in alphabetical order
  - "Quick list" of words and abbreviations
  - Detailed definitions with sample BASIC programs illustrating usage
- Machine language overview
  - Layman's introduction to using machine language with BASIC programs
- KERNAL
  - Overview of the KERNAL and its role in providing system-call compatibility for programs
- Input/Output Programming
  - Hookup and programming information for peripherals: lightpens, joysticks, disk drives, cassette, printers, modems
  - I/O techniques and device-specific programming guidance
- Graphics
  - Sprites (hardware sprites), programmable characters, and bitmap/high-resolution graphics for animation and detailed images
- Sound
  - SID (Sound Interface Device) synthesis overview for creating music and sound effects
- Languages
  - Information about running CP/M and other high-level languages (soft-load options) in addition to BASIC
