# An Introduction to Programming C-64 Demos — Distribution & Contribution Policy

**Summary:** This Commodore 64 demo programming document is released into the public domain; mirroring, copying, converting formats, and redistribution (including sale) are permitted. Source is available in LaTeX (.tex) and distributed formats (DVI, PostScript, HTML, ASCII); contributions are accepted as full-source or diffs (patches using diff).

**Copying and Mirroring**
The document is released into the public domain—you may mirror, copy, modify, convert to other formats, and redistribute it without restriction. If you upload or mirror the document, please notify the author so the mirror URL can be listed alongside the original.

Author homepage (original location): http://user.tninet.se/~uxm165t/

**Available formats**
The author provides the document in these formats:
- LaTeX source (.tex)
- DVI
- PostScript
- HTML
- Plain ASCII text

Conversion to other formats is explicitly encouraged.

**Contributing changes**
If you want to contribute edits or improvements:
- Preferred: modify the LaTeX source and send the revised full source.
- Alternative: send a patch/diff generated with the diff command (standard on Unix systems).

Include enough information to apply your changes (which file(s) were modified and how).
