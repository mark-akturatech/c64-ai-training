# Commodore 64 — Run CP/M on Z-80 Cartridge

**Summary:** Steps to start the Commodore CP/M(R) supplied with the Z-80 cartridge on a C64: LOAD the CP/M program from disk, execute RUN, and press RETURN. Searchable terms: Z-80 cartridge, CP/M(R), LOAD, RUN, RETURN.

## Procedure
1. LOAD the CP/M(R) program from your disk drive.
2. Type RUN.
3. Hit the <RETURN> key.
