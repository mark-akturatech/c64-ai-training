# Kick Assembler — Appendix A.4: Assembler Directives (Table A.7)

**Summary:** Quick reference for Kick Assembler directives and their purpose: origin operator (*), alignment (.align), assertions (.assert / .asserterror), breakpoints (.break), byte/word/dword output (.by/.byte/.dw/.dword), constants (.const), CPU selection (.cpu), macro/function blocks (.define), and disk image creation (.disk). Includes directive names and short descriptions relevant to assembly and output generation.

## Directives
This chunk is a compact list of assembler directives documented in Appendix A.4 (Table A.7). Each entry below names a directive and gives a concise description (examples from the original table are stored in the Source Code section).

- * — sets the current assembly origin (memory position).
- .align — align the current origin to the given value.
- .assert — assert that two expressions or codeblocks are equal.
- .asserterror — assert that a given expression or codeblock produces an error.
- .break — place a breakpoint at the next generated bytes.
- .by — alias for .byte.
- .byte — emit raw byte values.
- .const — define a constant name/value.
- .cpu — change the assembler instruction set (e.g. switch to _65c02).
- .define — execute a block of directives in functionmode (faster), typically used to define values or macros.
- .disk — create a .d64 disk image file (takes disk parameters and file parameters).
- .dw — alias for .dword.
- .dword — emit 4-byte (doubleword) values.

## Source Code
```text
Directive          Example                         Description

*                  *=$1000                         Sets the memory position to a given
                                                    value.

.align             .align $100                     Aligns the memory position with
                                                    the given value. Ex. '.align $100' at
                                                    memory position $1234 will set the
                                                    position to $1300.

.assert            .assert "Test 1",2+2,4           Asserts that two expressions or codeblocks are equal.

.asserterror       .asserterror "Test 2", List().get(27)
                                                    Asserts that a given expression or
                                                    codeblock generates an error.

.break             .break                          Puts a breakpoint on the next generated bytes.

.by                .by $01,$02,$03                 An alias for '.byte'.

.byte              .byte $01,$02,$03               Outputs bytes.

.const             .const DELAY=7                  Defines a constant.

.cpu               .cpu _65c02                     Changes the instruction set to a different cpu.

.define            .define width, height {...}     Executes a block of directives in
                                                    'functionmode' (faster) to define values.

.disk              .disk [..disk pararamters..] {..filepa-
                   rameters..}                      Creates a d64 image file.

.dw                .dw $12341234                    An alias for '.dword'.

.dword             .dword $12341234                 Outputs doublewords (4 byte values).
```
