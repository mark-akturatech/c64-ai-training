# .END — End-of-file pseudo-opcode

**Summary:** The .END pseudo-opcode is an assembler directive used as an end-of-file marker in assembler source listings to mark the end of the assembly input. Searchable terms: .END, assembler directive, end-of-file marker.

## Description
The .END directive denotes the end of the assembler input file or source listing. It functions as an end-of-file marker in assembler source listings to indicate that no further assembly source follows.
