# PETSCII Reverse-Video Mapping: $80-$9A (Dec 128–154)

**Summary:** PETSCII reverse-video character codes for decimal 128–154 (hex $80–$9A), showing the reverse variants of @ and A/a through Z/z. The "Reverse of" column lists the base PETSCII codes ($00–$1A) whose high-bit set forms the reverse characters (add $80).

## Reverse-video mapping
This table enumerates the PETSCII reverse-video characters in the C64 range $80–$9A (decimal 128–154). Each entry is the reverse (inverse) glyph corresponding to the base PETSCII code shown in the "Reverse of" column. (See referenced "reverse_video_intro" for the explanation that reverse characters are formed by adding $80 to the base code.)
