# Commodore 64 — Appendix I: Pinouts for Input/Output Devices

**Summary:** Appendix I lists the C64 external connector categories and pinout topics: Game I/O (joystick/paddle control ports), Cartridge Slot, Audio/Video, Serial I/O (disk/printer), RF Modulator Output, Cassette, and User Port.

## Overview
This appendix enumerates the external connector groups for the Commodore 64 and indicates which pinout diagrams are provided elsewhere. It is a navigational index rather than a full pin-level listing.

## Connector categories
1. Game I/O  
2. Cartridge Slot  
3. Audio/Video  
4. Serial I/O (Disk/Printer)  
5. Modulator Output (RF)  
6. Cassette  
7. User Port
