# BASIC Numeric Work Area ($57-$60)

**Summary:** Zero-page numeric workspace at $57-$60 (decimal 87–96). A busy, multi-use BASIC Numeric Work Area used by many BASIC routines.

## Description
This zero-page range ($57–$60) is reserved as the BASIC Numeric Work Area — a small, multi-purpose workspace used by numerous BASIC routines for temporary numeric storage and intermediate calculations. (zero-page workspace)

## Key Registers
- $0057-$0060 - BASIC - Numeric Work Area (zero-page temporary numeric workspace)
