# 6502 Instructions — Arithmetic Group

**Summary:** Group header for the 6502 arithmetic instructions ADC (Add with Carry) and SBC (Subtract with Carry); intended as a section/title node linking to full instruction details.

## Description
This node is a categorical header used in an instruction summary to group the two arithmetic mnemonics ADC and SBC. It contains no opcode encodings, addressing-mode lists, flag effects, or timing — those details are in the referenced instruction chunks.
