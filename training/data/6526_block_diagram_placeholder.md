# 6526 BLOCK DIAGRAM

**Summary:** The MOS Technology 6526 Complex Interface Adapter (CIA) is a versatile I/O chip used in the Commodore 64, providing parallel and serial I/O capabilities, timers, a time-of-day clock, and interrupt control. The following ASCII diagram represents the internal block structure of the 6526 CIA, detailing its functional components and their interconnections.

This diagram illustrates the primary functional blocks within the 6526 CIA:

- **Parallel I/O Ports:** Two 8-bit bidirectional ports (Port A and Port B) for general-purpose input/output operations.
- **Data Direction Registers (DDR):** Each port has a corresponding DDR (DDR A and DDR B) to configure the direction (input or output) of each pin.
- **Timers:** Two independent 16-bit interval timers (Timer A and Timer B) for timing and counting applications.
- **Time-of-Day Clock:** A real-time clock providing timekeeping functions.
- **Serial Shift Register:** An 8-bit register for serial data input and output operations.
- **Interrupt Control:** Manages interrupt requests generated by the various functional blocks.
