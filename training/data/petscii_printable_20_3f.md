# PETSCII Printable Characters $20-$3F (Dec 32-63)

**Summary:** PETSCII codes $20-$3F (decimal 32–63) are the printable ASCII-like range on the C64 covering space, punctuation, digits 0–9, and symbols : ; < = > ?. In standard PETSCII-to-Screen conversion these codes map directly to Screen Code $20-$3F.

## Character range
This chunk lists the PETSCII printable characters in the hex range $20–$3F (decimal 32–63). It covers:
- Space, punctuation and common symbols
- Digits 0–9
- Symbols : ; < = > ?

Note: the original column header "(up/gfx, lo/up)" refers to PETSCII's alternate character-set variants (uppercase/graphics vs lowercase/uppercase).
