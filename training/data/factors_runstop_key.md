# MACHINE — User note for FACTORS

**Summary:** Warning that the FACTORS program (machine language) can take a long time on large factors or prime numbers; the RUN/STOP key remains active to abort execution.

## Note
Even at machine language speeds, this program can take a long time to analyze large factors and prime numbers. The RUN/STOP key is active to allow the user to stop the run.
