# Increment and Decrement Group

**Summary:** 6502 instruction group containing INC, INX, INY, DEC, DEX, DEY — increment/decrement operations on memory and registers (INC/DEC: memory; INX/DEX/INY/DEY: X/Y registers). See individual instruction chunks for addressing modes, opcodes, and flag effects.

**Overview**

This section header groups the six 6502 increment/decrement instructions:

- **INC** (Increment Memory)
- **INX** (Increment X Register)
- **INY** (Increment Y Register)
- **DEC** (Decrement Memory)
- **DEX** (Decrement X Register)
- **DEY** (Decrement Y Register)

Use the referenced instruction chunks for full details: opcode values, addressing modes, exact flag behavior, cycle counts, and example code.
