# ca65 -S, --segment-list

**Summary:** ca65 assembler option (-S / --segment-list) — prepend the segment ID to each address shown in the generated listing file; affects listing file content and formatting (see listing_option).

## Description
Add the segment ID in front of each address in the listing file. The option is available in short form (-S) and long form (--segment-list). It changes how addresses are rendered in the assembler's listing output by including the segment identifier with every address entry.
