# ca65 -V, --version

**Summary:** ca65 assembler command-line option `-V` / `--version` prints the assembler version number; useful when reporting bugs or sending suggestions.

## Description
The `-V` or `--version` switch causes ca65 to output its version number and exit. Include this reported version number when filing bug reports or sending suggestions so developers can reproduce issues against the correct assembler release.
