# Commodore Publications & Support — POWER/PLAY, COMMODORE magazine, Users' Clubs, VICMODEM networks

**Summary:** Describes Commodore's consumer and technical publications (POWER/PLAY, COMMODORE magazine), modem/network services (VICMODEM, The Source, CompuServe), and support channels including Commodore Information Network and local Users' Clubs.

**Overview**
Commodore provided post‑sale support via two primary publications, an online/two‑way information network for U.S. and Canada users, and encouragement of independently run Commodore Users' Clubs. Typical applications highlighted include simulations, sports data, and stock quotes accessed with a VICMODEM and network subscriptions.

**POWER/PLAY (Home Computer Magazine)**
- **Target:** Home/consumer users of Commodore computers.
- **Content:** User club listings and news, software and game coverage, home learning applications, programming techniques, telecommunications, and new hardware/software products.
- **Publication Frequency:** Quarterly.
- **Price:** $10.00 per year (as stated).
- **Subscription Contact:** To subscribe, write to: POWER/PLAY Magazine, P.O. Box 5406, Greensboro, NC 27403, USA. ([commodore.ca](https://www.commodore.ca/gallery/magazines/compute/Compute-034-03.pdf?utm_source=openai))

**COMMODORE magazine (Technical)**
- **Target:** Technical audience (developers, advanced users).
- **Content:** Technical material and Commodore‑oriented developer information.
- **Publication Frequency:** Monthly.
- **Price:** $9.97 per year.
- **Subscription Contact:** To subscribe, call toll-free 800-727-6937 or write to: COMMODORE Magazine, P.O. Box 3244, Harlan, IA 51593-2424, USA. ([commodore.ca](https://www.commodore.ca/gallery/magazines/compute/Compute-147-02.pdf?utm_source=openai))

**Online Services and VICMODEM Usage**
- **VICMODEM:** Commodore's modem hardware used to access online services.
- **Services Named:** The Source and CompuServe — examples given for retrieving sports data and stock quotes with a VICMODEM and appropriate subscription.
- **Commodore Information Network:** A "two‑way" computer information network providing input for users across the U.S. and Canada.
  - **Access Details:** Users could connect via the VICMODEM using standard telephone lines.
  - **Protocols:** Supported standard communication protocols of the time, such as XMODEM for file transfers.
  - **Bulletin Board Addresses:** Specific bulletin board phone numbers were provided in Commodore publications and user manuals.

**Users' Clubs and Local Dealer Support**
- **Users' Clubs:** Commodore advocated and supported independent Commodore Users' Clubs worldwide as a resource for both beginner and advanced users.
- **Local Dealers:** Local Commodore dealers are noted as additional support and information sources.
