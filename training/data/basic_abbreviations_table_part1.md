# Commodore 64 BASIC keyword abbreviation table — part 1

**Summary:** First page of the Commodore 64 BASIC keyword abbreviation table mapping full BASIC commands (ABS, AND, ASC, ATN, CHR$, CLOSE, CLR, CMD, CONT, COS, DATA, DEF, DIM, LEFT$, LEN, LET, etc., up through RND) to their on‑screen abbreviation and key sequence (e.g. "A <SHIFT+B>"). Shows the table column headers and exact "Looks like on screen" strings.

## Table description
This chunk is the first page of the BASIC keyword abbreviation table showing, for each BASIC command, the printed abbreviation and the exact key/shift sequence as it appears on the C64 screen. It preserves the original column headers and formatting and covers the commands listed up through RND.
