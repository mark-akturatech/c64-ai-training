# Commodore 64 BASIC — Abbreviation table continuation (LET … WAIT)

**Summary:** Continuation/completion of the C64 BASIC keyword abbreviation table listing BASIC keywords and their keyboard abbreviations (single-key, shifted-key combos, and two-letter abbreviations) and what appears on-screen when used. Includes entries from LET through WAIT as found on the continuation page.

## Abbreviation table continuation
This chunk is a page-continuation of the BASIC keyword abbreviation table showing each BASIC command, the abbreviated key or key-combination (if any) used to enter it, and the text that appears on-screen when the abbreviation is used. Entries include single-key abbreviations (often with SHIFT combinations), two-letter abbreviations (e.g., ST, RE, TI), explicit "NONE" where no abbreviation exists, and the special single-character PRINT abbreviation (?).
