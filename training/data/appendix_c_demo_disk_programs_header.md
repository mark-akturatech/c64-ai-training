# Appendix C: Demonstration Disk Programs

**Summary:** Container for demonstration disk utilities for the Commodore 64: DIR (directory listing), VIEW BAM (Block Availability Map), DISPLAY T & S (display Track & Sector), CHECK DISK, and PERFORMANCE TEST. Includes pointers to expanded example program chunks.

## Overview
This appendix groups the example disk programs provided with the demo disk. It serves as an index/introduction to the programs that follow:
- DIR — directory listing utility
- VIEW BAM — visualizer for the disk's Block Availability Map (BAM)
- DISPLAY T & S — shows raw Track & Sector contents
- CHECK DISK — disk consistency/check utility
- PERFORMANCE TEST — measures disk operation throughput
