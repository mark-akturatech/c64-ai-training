# Magazines with C-64 Demo Programming Tutorials

**Summary:** Listings and availability for three C‑64/demo programming magazines: C=Hacking (assembler tutorials, demo effects), Discovery (demo articles, ASCII issues), and Coders World (disk magazine with beginner code samples). Includes download locations (ftp.funet.fi, ffd2.com/fridge) and a note about a hard-to-find third issue.

**Overview**

**C=Hacking**

- Online magazine covering Commodore 8‑bit topics. Early issues include a multipart assembler tutorial and demo-effect articles by Pasi "Albert" Ojala.
- Issues are distributed as ASCII text files.

**Discovery**

- Short run (three issues located by the source) with demo-related articles and learnable techniques.
- Also distributed as ASCII text files.

**Coders World**

- Disk magazine (run on a real C‑64) with simple code examples and explanations aimed at beginners; contains demo-style effects and sample programs.
- Three issues were released; the material is more introductory and may be basic for experienced demo coders.

**Availability**

- **C=Hacking (ASCII text):** [ftp://ftp.funet.fi/pub/cbm/magazines/c=hacking/](ftp://ftp.funet.fi/pub/cbm/magazines/c=hacking/)
- **Discovery (ASCII text):** [http://www.ffd2.com/fridge/discovery/](http://www.ffd2.com/fridge/discovery/)
- **Coders World (disk mag):** First two issues available at [ftp://ftp.funet.fi/pub/cbm/magazines/disk/c64/](ftp://ftp.funet.fi/pub/cbm/magazines/disk/c64/); third issue noted as "hard to find" by the source.

**Issue Details**

**C=Hacking**

- **Issue #5 (March 1993):** Includes "Stretching Sprites" by Pasi 'Albert' Ojala. [Source](https://csbruce.com/cbm/hacking/)
- **Issue #6 (September 1993):** Features "DYCP - Horizontal Scrolling" and "Opening the borders" by Pasi 'Albert' Ojala. [Source](https://csbruce.com/cbm/hacking/)
- **Issue #7 (February 1994):** Contains "FLD - Scrolling the Screen" and "Tech-tech - more resolution to vertical shift" by Pasi 'Albert' Ojala. [Source](https://csbruce.com/cbm/hacking/)

**Discovery**

- **Issue #1:** Articles on demo-related techniques.
- **Issue #2:** Further demo programming insights.
- **Issue #3:** Additional demo articles and tutorials.

**Coders World**

- **Issue #1:** Introductory code examples and explanations.
- **Issue #2:** More beginner-friendly demo-style effects and sample programs.
- **Issue #3:** Content details are scarce; noted as hard to find.
