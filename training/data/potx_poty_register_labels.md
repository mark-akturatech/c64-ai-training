# POTX / POTY — $D419 / $D41A (SID paddle inputs)

**Summary:** POTX ($D419) and POTY ($D41A) are SID (MOS6581) registers used to read analog game paddle positions: POTX for Game Paddle 1 (or 3) and POTY for Game Paddle 2 (or 4). Proper coordination between the SID and CIA is required to obtain valid readings.

**Description**

POTX and POTY are SID-addressed paddle input registers that provide the current position of the connected analog paddles. The source mapping labels:

- $D419 — POTX — Read Game Paddle 1 (or 3) Position
- $D41A — POTY — Read Game Paddle 2 (or 4) Position

The registers are read-only and return values representing paddle positions. Reading paddles on the C64 requires coordination between the SID paddle inputs and CIA port A switching.

**Procedure for Sampling Paddles:**

1. **Configure CIA Port A:**
   - Set the data direction register of CIA port A ($DC02) to output mode for the relevant bits.
   - Output a logic low on the desired bit of CIA port A ($DC00) to select the paddle to be read.

2. **Initiate SID Conversion:**
   - The SID measures the analog input once every 512 machine cycles. Ensure that sufficient time has elapsed since the last read to allow the SID to complete the conversion.

3. **Read Paddle Position:**
   - Read the POTX ($D419) or POTY ($D41A) register to obtain the paddle position. The value ranges from 0 (fully clockwise) to 255 (fully counter-clockwise).

**Data Format:**

- **Bit Width:** 8 bits
- **Value Scaling:** 0 to 255, where 0 represents the paddle turned fully clockwise and 255 fully counter-clockwise.

**SID Register Map Context:**

- **$D400-$D418:** Voice and filter control registers
- **$D419:** POTX — Read Game Paddle 1 (or 3) Position
- **$D41A:** POTY — Read Game Paddle 2 (or 4) Position
- **$D41B-$D41C:** Unused
- **$D41D:** SID Random Number Generator
- **$D41E-$D41F:** Unused

**Adjacent SID Registers:**

- **$D41D — Random Number Generator:**
  - **Bit 7-0:** Random value generated by the SID.

## Source Code

```assembly
    LDA #$FE        ; Set bit 0 of CIA port A low to select paddle 1
    STA $DC00       ; Write to CIA port A
    LDA $D419       ; Read POTX register
    ; A now contains the paddle position (0-255)
```

```basic
10 POKE 56320, 254  : REM Set bit 0 of CIA port A low to select paddle 1
20 X = PEEK(54297)  : REM Read POTX register
30 PRINT "Paddle position: "; X
```
