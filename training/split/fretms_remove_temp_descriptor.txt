# ********* - FRETMS: Remove an entry from the string descriptor stack. If a currently valid string's descriptor matches an entry on the temporary string descriptor stack, that stack entry is removed to keep descriptor bookkeeping consistent.


46811         $B6DB          FRETMS
Remove an Entry from the String Descriptor Stack

If the descriptor of a currently valid string is the same as one of
the entries on the temporary string descriptor stack, the stack entry
is removed.

---
Additional information can be found by searching:
- "frestr_discard_temporary_string" which expands on FRESTR calls this routine to clear temp descriptors
