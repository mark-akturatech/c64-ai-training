# Kick Assembler - Describes .disk directive format: .disk OPT_WRITERNAME [...DISK PARAMETERS..] { [..FILE1 PARAMETERS..], [..FILE2 PARAMETERS..], ... } The writer name is optional: default writer used otherwise plugin writer looked up. Default writer based on CC1541; plugins can supply other writers.

Description

filename="MyDisk.d64"

The name of the disk image

format

"commodore"

format="commodore"

Sets the format of the disk.
