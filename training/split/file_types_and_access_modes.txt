# Commodore IEC Serial Bus Protocol Reference - pagetable.com - DOS file types and access modes: Sequential file types SEQ/PRG/USR. Access modes R (read), M (recovery), W (write), A (append).

File Types:
  Sequential files support three types:
    SEQ - Sequential data storage
    PRG - Executable programs
    USR - User-defined data

  Access modes:
    R - Read
    M - Recovery mode
    W - Write
    A - Append
