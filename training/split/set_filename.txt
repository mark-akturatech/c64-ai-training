# - Fully Commented Commodore 64 ROM Disassembly (English) - Set filename pointer and length for I/O operations: STA $B7 (length), STX $BB (pointer low), STY $BC (pointer high). RTS.

                                *** set filename
.,FDF9 85 B7    STA $B7         set file name length
.,FDFB 86 BB    STX $BB         set file name pointer low byte
.,FDFD 84 BC    STY $BC         set file name pointer high byte
.,FDFF 60       RTS
