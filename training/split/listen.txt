# Commodore 64 KERNAL Functions Reference - LISTEN ($FFB1). Sends a LISTEN command to a device on the serial IEC bus. Input: A = device number. Uses A register. Real address: $ED0C.

$FFB1 - LISTEN
Sends LISTEN command to serial bus. Input: A = device number. Uses A register. Real address: $ED0C.
