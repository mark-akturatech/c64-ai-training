# - Fully Commented Commodore 64 ROM Disassembly (English) - Unpack the memory operand pointed to by (AY) into FAC2 (JSR $BA8C). This small segment is the entry that prepares FAC2 when adding a memory operand to FAC1.


                                *** add (AY) to FAC1
.,B867 20 8C BA JSR $BA8C       unpack memory (AY) into FAC2
