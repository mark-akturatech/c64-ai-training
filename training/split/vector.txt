# Commodore 64 KERNAL Functions Reference - VECTOR ($FF8D). Transfers the interrupt vector table between system and user areas. Input: Carry flag (0 = copy user to system, 1 = copy system to user); X/Y = pointer to user table. Uses registers A, Y. Real address: $FD1A.

$FF8D - VECTOR
