# Commodore 64 Complete Memory Map - Source: sta.c64.org - System Area $02A1-$02FF: CIA interrupt/backups ($02A1-$02A4), scroll line count ($02A5), PAL/NTSC switch ($02A6), and remaining unused bytes up to $02FF.

$02A1   CIA2 Interrupt Backup   CIA#2 interrupt control register backup
$02A2   CIA1 Timer A Backup     CIA#1 timer A control register backup
$02A3   CIA1 Interrupt Backup   CIA#1 interrupt control register backup
$02A4   CIA1 Timer B Backup     CIA#1 timer B control register backup
$02A5   Scroll Line Count       Current line number being scrolled
$02A6   PAL/NTSC Switch         System video standard (0=NTSC, 1=PAL)
$02A7-$02FF  Unused             (89 bytes)
