# Commodore 64 Complete Memory Map - Source: sta.c64.org - Expansion I/O Areas $DE00-$DFFF: optional external device registers mapped as I/O Area 1 ($DE00-$DEFF) and I/O Area 2 ($DF00-$DFFF), each 256 bytes for cartridges or expansion hardware.

Expansion I/O Areas ($DE00-$DFFF)
----------------------------------

$DE00-$DEFF  I/O Area 1         Optional external device registers (256 bytes)
$DF00-$DFFF  I/O Area 2         Optional external device registers (256 bytes)
