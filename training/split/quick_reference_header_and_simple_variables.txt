# COMMODORE 64 - Header and SIMPLE VARIABLES: Commodore 64 Quick Reference Card title and concise description of simple variable types including Real, Integer (%) and String ($), their numeric/character ranges, and variable-naming rules (X = letter, Y = letter/number; only first two characters are significant).

~


                      COMMODORE 64 QUICK REFERENCE CARD

  SIMPLE VARIABLES

  Type     Name     Range

  Real     XY       +-1.70141183E+38
                    +-2.93873588E-39
  Integer  XY%      +-32767
  String   XY$      0 to 255 characters

    X is a letter (A-Z), Y is a letter or number (0-9). Variable names
  can be more than 2 characters, but only the first two are recognized.
