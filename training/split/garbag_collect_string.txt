# ********* - Subroutine: Collect a String. Moves a string's text up to high memory (toward the bottom of string storage) and updates the string descriptor to point to the new location.

46598         $B606
Collect a String

This part of the garbage collection routine moves the string to high
memory and updates the descriptor to point to its new location.

---
Additional information can be found by searching:
- "garbag_check_most_eligible" which expands on first determines which strings are eligible before collection
- "movins_move_string" which expands on MOVINS is the utility often used to move string bytes when collecting
