# 6502 Microprocessor - CMP pseudocode: compute AC - src, set Carry if no borrow (src < 0x100), set Sign and Zero based on low byte result.

/* CMP */
    src = AC - src;
    SET_CARRY(src < 0x100);
    SET_SIGN(src);
    SET_ZERO(src &= 0xff);
