# Commodore 64 KERNAL Functions Reference - MEMTOP ($FF99). Manages BASIC workspace upper boundary. Input: Carry (0 = restore, 1 = save); if Carry=0 then X/Y = address to restore; if Carry=1 then X/Y returned = saved address. Uses X, Y registers. Real address: $FE25.

$FF99 - MEMTOP
Manages BASIC workspace upper boundary. Input: Carry (0=restore, 1=save); X/Y = address if Carry=0. Output: X/Y = address if Carry=1. Uses X, Y registers. Real address: $FE25.
