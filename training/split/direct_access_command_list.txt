# byRiclianll - List of nine 1541 direct-access commands and short functions: Block-Read (U1) - read block into 1541 RAM; Buffer-Pointer (B-P) - set pointer to any byte in disk buffer; Block-Write (U2) - write buffer to disk; Memory-Read (M-R) - peek bytes out of 1541 RAM/ROM; Memory-Write (M-W) - poke bytes into 1541 RAM; Block-Allocate (B-A) - set bit in BAM to indicate sector in use; Block-Free (B-F) - clear BAM bit; Memory-Execute (M-E) - execute 6502 routine in 1541 RAM/ROM; Block-Execute (B-E) - load & execute 6502 routine in 1541 RAM.

The  1541  DOS  recognizes  nine  direct-access  commands.  These  direct-access  commands 
and  their  functions  are  listed  below. 


Direct-Access  Command  Function 


Block-Read  (Ul)  Read  a  data  block  into  1541  RAM. 

Buffer-Pointer  (B-P)  Set  pointer  to  any  byte  in  a  disk  buffer. 

Block- Write  (U2)  Write  a  data  block  from  1541  RAM  to  diskette.
