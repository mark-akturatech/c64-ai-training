# Commodore 64 Complete Memory Map - Source: sta.c64.org - Processor Stack region header ($0100-$01FF) and contents: $0100-$013D error bytes (datasette pointers) and the system processor stack area used for subroutine/interrupt stacking and FOR/GOSUB storage ($0100-$01FF).

Processor Stack ($0100-$01FF)
-----------------------------

$0100-$013D  Error Bytes        Pointers to error bytes from datasette (62 bytes)
$0100-$01FF  Stack Area         System processor stack and FOR/GOSUB storage
