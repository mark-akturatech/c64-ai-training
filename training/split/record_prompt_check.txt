# - Fully Commented Commodore 64 KERNAL ROM Disassembly (English, "CBM") - CSTE2 routine: checks for both PLAY and RECORD being active and, if so, triggers the display of the 'RECORD' message (MSG). Branches back to the 'PRESS PLAY' path if necessary.

                                ;CHECKS FOR PLAY & RECORD
                                ;
.,F838 20 2E F8 JSR $F82E       CSTE2  JSR CS10
.,F83B F0 F9    BEQ $F836       BEQ    CS25
.,F83D A0 2E    LDY #$2E        LDY    #MS8-MS1        ;"RECORD"
.,F83F D0 DD    BNE $F81E       BNE    CS30
