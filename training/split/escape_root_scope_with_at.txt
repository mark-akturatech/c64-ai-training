# Kick Assembler - Explains escaping the current scope using the '@' prefix to reference the root scope. Example: .label myLabel = 1 { .label myLabel = 2 .print "scoped myLabel="+ myLabel (.-> 2) .print "root myLabel="+ @myLabel (.-> 1) }

Or for functions, macros or pseudo commands, here shown in a library file:
#import "mylib.lib"
.print myFunction()
MyMacro()
MyPseudoCommand

/* File mylib.lib */
#importonce
.filenamespace MyLibrary
