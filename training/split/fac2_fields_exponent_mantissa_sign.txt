# ********* - FAC2 subfields: ARGEXP ($69) — exponent for FAC2; ARGHO ($6A-$6D) — mantissa bytes for FAC2; ARGSGN ($6E) — sign byte for FAC2. The format is the same as FAC1.

105           $69            ARGEXP
Floating Point Accumulator #2: Exponent

106-109       $6A-$6D        ARGHO
Floating Point Accumulator #2: Mantissa

110           $6E            ARGSGN
Floating Point Accumulator #2: Sign
