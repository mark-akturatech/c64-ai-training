# Commodore 64 KERNAL Functions Reference - RESTOR ($FF8A). Restores the default interrupt vector table at $0314-$0333. No input/output parameters. Real address: $FD15.

$FF8A - RESTOR
Restores default interrupt vector table at $0314-$0333. No parameters. Real address: $FD15.
