# - Fully Commented Commodore 64 ROM Disassembly (English) - Entry check for FAC1 == 0. Branches if FAC1 is non-zero; if FAC1 is zero jump to routine that copies FAC2 into FAC1 and returns.

                                *** add FAC2 to FAC1
.,B86A D0 03    BNE $B86F       branch if FAC1 is not zero
.,B86C 4C FC BB JMP $BBFC       FAC1 was zero so copy FAC2 to FAC1 and return
                                FAC1 is non zero
