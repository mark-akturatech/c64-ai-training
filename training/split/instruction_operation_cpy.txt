# 6502 Microprocessor - CPY pseudocode: YR - src, set Carry if no borrow, set Sign and Zero based on result.

/* CPY */
    src = YR - src;
    SET_CARRY(src < 0x100);
    SET_SIGN(src);
    SET_ZERO(src &= 0xff);
