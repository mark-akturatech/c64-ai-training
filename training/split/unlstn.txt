# Commodore 64 KERNAL Functions Reference - UNLSTN ($FFAE). Sends the UNLISTEN command on the serial IEC bus to end a LISTEN session. Uses A register. Real address: $EDFE.

$FFAE - UNLSTN
Sends UNLISTEN command to serial bus. Uses A register. Real address: $EDFE.
