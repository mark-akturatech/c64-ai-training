# ********* - SCRTCH (NEW): implements the NEW command by storing two zeros in the link address of the first program line to mark end-of-program and setting the end-of-program pointer ($2D-$2E) to the byte past those zeros. Execution continues into CLR to complete clearing operations.

42562         $A642          SCRTCH
Perform NEW

The NEW command stores two zeros in the link address of the first
program line to indicate the end of program, and sets the end of
program pointer at 45-46 ($2D-$2E) to point to the byte past those
zeros.  It continues through to the CLR command code.
