# ********* - ROPRTY ($BD) — Used as an output-parity work byte by the RS-232 routines and as temporary storage for the current character being read or sent by the cassette/tape routines.

189           $BD            ROPRTY
RS-232 Output Parity/Cassette Temporary Storage

This location is used by the RS-232 routines as an output parity work
byte, and by the tape as temporary storage for the current character
being read or sent.
