# Commodore 64 KERNAL Functions Reference - CLRCHN ($FFCC). Closes default I/O files, sends UNTALK/UNLISTEN as needed on the serial bus, and restores keyboard/screen I/O. Uses A, X registers. Real address: indirect via ($0322) then $F333.

$FFCC - CLRCHN
Closes default I/O files, sends UNTALK/UNLISTEN as needed, restores keyboard/screen. Uses A, X registers. Real address: ($0322), $F333.


---
Additional information can be found by searching:
- "untalk" which expands on CLRCHN issues UNTALK when needed (UNTALK $FFAB)
- "unlstn" which expands on CLRCHN issues UNLISTEN when needed (UNLSTN $FFAE)
