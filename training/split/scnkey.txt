# Commodore 64 KERNAL Functions Reference - SCNKEY ($FF9F). Scans the keyboard matrix: stores raw code at $00CB, shift status at $028D, and converts to PETSCII code written to buffer. Uses A, X, Y registers. Real address: $EA87.

$FF9F - SCNKEY
Scans keyboard matrix, stores code at $00CB, shift status at $028D, PETSCII code to buffer. Uses A, X, Y registers. Real address: $EA87.
