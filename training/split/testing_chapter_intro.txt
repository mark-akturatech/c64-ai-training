# Kick Assembler - Chapter 16 Testing: Kick Assembler provides .assert directives useful for testing assembler features, macros, pseudo-commands, and functions. Assembling counts assertions and failed assertions and reports at the end.

Assembler website.
To use it do the following:
1. Create an Eclipse workspace.
2. ’Import->Existing Projects into workspace->Select archive file’ and select the downloaded project file.
3. Replace the KickAss.jar file in the jars folder with the newest version, if necessary.
You are now ready to start. In the src folder you can see examples of how the plugins are made. The files in
PluginTest shows how to use them and in the launch folder is launch files for running the examples (Rightclick>Run As).
