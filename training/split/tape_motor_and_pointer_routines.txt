# ********* - Tape-related Kernal routines: 'Turn Off the Tape Motor' (heading), 'Check the Tape Read/Write Pointer' (compares current and ending tape read/write addresses), and 'Advance the Tape Read/Write Pointer' (increments the current read/write pointer by one byte).

Turn Off the Tape Motor

64721         $FCD1
Check the Tape Read/Write Pointer

This routine compares the current tape read/write address with the
ending read/write address.

64731         $FCDB
Advance the Tape Read/Write Pointer

This routine is used to move the pointer to the current read/write
address up a byte.
