# ********* - CHRD (CHR$): Creates a one-byte string descriptor on the temporary string stack for the specified byte value and sets a pointer to that temporary string.


46828         $B6EC          CHRD
Perform CHR$

The CHR$ routine creates a descriptor on the temporary string stack
for the one-byte string whose value is specified in the command, and
sets a pointer to that string.

---
Additional information can be found by searching:
- "frestr_discard_temporary_string" which expands on temporary descriptors created by CHR$ are cleared by FRESTR/FRETMS when discarded
