# ********* - Explains RINONE ($A9), the RS-232 flag used to detect a start bit. A value of 144 ($90) indicates no start bit was received; 0 indicates a start bit was received.

169           $A9            RINONE
RS-232 Flag: Check for Start Bit

This flag is used when checking for a start bit.  A 144 ($90) here
indicates that no start bit was received, while a 0 means that a start
bit was received.
