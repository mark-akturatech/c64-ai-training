# - Fully Commented Commodore 64 ROM Disassembly (English) - Copy I/O start address to tape buffer address: LDA $C2 -> STA $AD (buffer high), LDA $C1 -> STA $AC (buffer low), RTS. Used to set buffer pointer before tape read/write operations.

.,FB94 85 AC    STA $AC         set buffer address low byte
.,FB96 60       RTS
