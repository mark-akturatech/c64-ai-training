# Commodore 64 KERNAL Functions Reference - STOP ($FFE1). Queries the STOP key status at $0091; if pressed, calls CLRCHN and clears input buffer. Output: Zero flag (0 = unpressed, 1 = pressed); Carry flag (1 = pressed). Uses A, X registers. Real address: indirect via ($0328) then $F6ED.

$FFE1 - STOP
Queries STOP key status at $0091; if pressed, calls CLRCHN and clears buffer. Output: Zero flag (0=unpressed, 1=pressed); Carry flag (1=pressed). Uses A, X registers. Real address: ($0328), $F6ED.


---
Additional information can be found by searching:
- "clrchn" which expands on CLRCHN is called when STOP is detected (CLRCHN $FFCC)
- "udtim" which expands on UDTIM updates STOP key indicator (UDTIM $FFEA)
