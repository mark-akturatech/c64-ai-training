# Commodore 64 Complete Memory Map - Source: sta.c64.org - KERNAL ROM region $E000-$FFFF: 8 KB operating system kernel ROM or RAM if banked out. Visibility controlled by processor port $0001 bits #0-#2. Contains OS service routines referenced by BASIC and machine code.

KERNAL ROM ($E000-$FFFF)
--------------------------

$E000-$FFFF  KERNAL ROM/RAM     Operating system kernel (8192 bytes)

Visibility controlled by processor port $0001 bits #0-#2.
