# ********* - LEFTD (LEFT$): Creates a temporary string descriptor for a new string consisting of a specified number of characters taken from the left side of a source string.


46848         $B700          LEFTD
Perform LEFT$

LEFT$ creates a temporary string descriptor for a new string which
contains the number of characters from the left side of the string
that is specified in the command.
