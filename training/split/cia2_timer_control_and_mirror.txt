# Commodore 64 Complete Memory Map - Source: sta.c64.org - CIA #2 timer control $DD0E-$DD0F (Timer A/B control) and CIA#2 image mirror area $DD10-$DDFF (register images repeated every $10 bytes).

Timer Control:

$DD0E   Timer A Control         Start, output mode, and count source
$DD0F   Timer B Control         Start, output mode, and count source

Mirror Area:

$DD10-$DDFF  CIA#2 Images       Register images (repeated every $10 bytes)
