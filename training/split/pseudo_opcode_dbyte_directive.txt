# Contents - Documentation of the .DBYTE pseudo-opcode: reserves 16-bit data in HIGH byte-LOW byte format (the opposite byte order from WORD). This section contains the .DBYTE label and its concise description as used in the assembler listings.


.DBYTE 


Reserves 16 bit data in a 




HIGH byte-LOW byte format
