# 6502 Microprocessor - CPX pseudocode: XR - src, set Carry if no borrow, set Sign and Zero based on result.

/* CPX */
    src = XR - src;
    SET_CARRY(src < 0x100);
    SET_SIGN(src);
    SET_ZERO(src &= 0xff);
