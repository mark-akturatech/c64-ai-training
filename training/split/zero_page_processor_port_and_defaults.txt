# Commodore 64 Complete Memory Map - Source: sta.c64.org - Zero Page addresses $0000-$0006: $0000 Processor Port DDR (data direction for processor port bits), $0001 Processor Port (memory area configuration and datasette control), $0002-$0004 and $0005-$0006 unused-default execution addresses for floating-point and integer conversion routines.

$0000   Processor Port DDR      Data direction for processor port bits
$0001   Processor Port          Memory area configuration and datasette control
$0002-$0004  Unused             Default execution addresses for FP conversion routines
$0005-$0006  Unused             Default execution addresses for integer conversion
