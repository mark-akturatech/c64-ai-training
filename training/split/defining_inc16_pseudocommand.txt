# Kick Assembler - Defines a 16-bit increment pseudo command .pseudocommand inc16 arg that increments the low byte and, on carry (BNE/over logic), increments the high/next argument using _16bitnextArgument(arg).

.pseudocommand inc16 arg {
inc arg
bne over
inc _16bitnextArgument(arg)
over:
}
