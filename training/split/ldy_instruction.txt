# 6502 Instructions summary - LDY (Load Y Register). Addressing modes and opcodes: Absolute LDY $aaaa ($AC, 3 bytes) sets N,Z; Zero Page LDY $aa ($A4, 2) ; Immediate LDY #$aa ($A0, 2) ; Absolute Indexed X LDY $aaaa,X ($BC, 3) ; Zero Page Indexed X LDY $aa,X ($B4, 2). Flags affected: Negative, Zero.

LDY	Load Y Register	Absolute	LDY $aaaa	$AC	3	N,Z
		Zero Page	LDY $aa	$A4	2	
		Immediate	LDY #$aa	$A0	2	
		Absolute Indexed, X	LDY $aaaa,X	$BC	3	
		Zero Page Indexed, X	LDY $aa,X	$B4	2
