# MACHINE - Instruction snippet showing how to change the Start-Of-Variables pointer on VIC-20: display SOV with '.M 002D 002E' and change the pointer bytes; example shows editing to place pointer above the machine language program (hex C8 10 ...).

To change the start-of-variables pointer to a location above the machine
language program, display the SOV pointer with .M 002D 002E and change the
pointer to

  .:002D C8 10 .. .. .. .. .. ..
