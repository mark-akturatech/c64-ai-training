# COMMODORE 64 - LEN(string): returns number of characters in the string, counting blanks and non-printed characters. Example showing LEN on 'COMMODORE COMPUTER' returning 18.








  56   BASIC LANGUAGE VOCABULARY
~


  LEN

  TYPE: Integer Function
  Format: LEN (<string>)

    Action: Returns the number of characters in the string expression.
  Non-printed characters and blanks are counted.

  EXAMPLE of LEN Function:

    CC$ = "COMMODORE COMPUTER": PRINT LEN(CC$)

     18
