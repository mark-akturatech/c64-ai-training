# Commodore 64 Complete Memory Map - Source: sta.c64.org - Hardware vectors at $FFFA-$FFFF: $FFFA-$FFFB NMI vector (default $FE43), $FFFC-$FFFD Reset vector (cold reset entry, default $FCE2), $FFFE-$FFFF IRQ/BRK vector (default $FF48). These addresses contain the 16-bit pointers to the system handlers.

Hardware Vectors ($FFFA-$FFFF)
-------------------------------

$FFFA-$FFFB  NMI Vector         Non-maskable interrupt handler (default: $FE43)
$FFFC-$FFFD  Reset Vector       Cold reset entry point (default: $FCE2)
$FFFE-$FFFF  IRQ Vector         Interrupt request handler (default: $FF48)
