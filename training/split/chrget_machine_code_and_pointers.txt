# Commodore 64 Complete Memory Map - Source: sta.c64.org - Zero Page $0073-$007B: CHRGET machine code area ($0073-$008A covers code for reading BASIC bytes, entry at $0079), plus CHRGET pointer(s) ($007A-$007B) used to reference the current BASIC byte during tokenization and listing.

$0073-$008A  CHRGET             Machine code for reading BASIC bytes (24 bytes)
$0079   CHRGOT                  Entry point to read current byte
$007A-$007B  CHRGET Ptr         Pointer to current BASIC byte
