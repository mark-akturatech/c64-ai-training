# Commodore 64 KERNAL Functions Reference - CLOSE ($FFC3). Closes a specified file. Input: A = logical file number. Uses registers A, X, Y. Real address: indirect via ($031C) then $F291.

$FFC3 - CLOSE
Closes specified file. Input: A = logical number. Uses A, X, Y registers. Real address: ($031C), $F291.
