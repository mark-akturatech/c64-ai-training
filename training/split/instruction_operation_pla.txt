# 6502 Microprocessor - PLA pseudocode: pull a byte from stack into src, update Sign and Zero flags and set AC = pulled value.

/* PLA */
    src = PULL();
    SET_SIGN(src);	/* Change sign and zero flag accordingly. */
    SET_ZERO(src);
