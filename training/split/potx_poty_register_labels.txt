# - C64 I/O Map (Mapping The Commodore 64) - Register labels: $D419 POTX (Read game paddle 1 or 3), $D41A POTY (Read game paddle 2 or 4).

$D419        POTX         Read Game Paddle 1 (or 3) Position

$D41A        POTY         Read Game Paddle 2 (or 4) Position
