# - C64 RAM Map (Mapping The Commodore 64) - $029B RIDBE: RS-232 index to the end of the receive buffer. Points to the ending byte within the 256-byte receive buffer and is used to add data to that buffer.

$029B        RIDBE   RS-232: Index to End of Receive Buffer
                     
                     This index points to the ending byte within the 256-byte RS-232
                     receive buffer, and is used to add data to that buffer.
