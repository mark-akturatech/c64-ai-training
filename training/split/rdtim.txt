# Commodore 64 KERNAL Functions Reference - RDTIM ($FFDE). Reads the Time-of-Day (TOD) clock from $00A0-$00A2. Output: A/X/Y = current TOD value. Uses A, X, Y registers. Real address: $F6DD.

$FFDE - RDTIM
Reads Time of Day from $00A0-$00A2. Output: A/X/Y = current TOD value. Uses A, X, Y registers. Real address: $F6DD.


---
Additional information can be found by searching:
- "settim" which expands on set TOD (SETTIM $FFDB)
- "udtim" which expands on UDTIM updates TOD and STOP key indicator (UDTIM $FFEA)
