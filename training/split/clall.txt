# Commodore 64 KERNAL Functions Reference - CLALL ($FFE7). Clears the file table and calls CLRCHN to close channels and restore I/O. Uses A, X registers. Real address: indirect via ($032C) then $F32F.

$FFE7 - CLALL
Clears file table and calls CLRCHN. Uses A, X registers. Real address: ($032C), $F32F.


---
Additional information can be found by searching:
- "clrchn" which expands on CLRCHN performs channel close and UNLISTEN/UNTALK (CLRCHN $FFCC)
