# Commodore 64 KERNAL Functions Reference - UDTIM ($FFEA). Updates the Time-of-Day ($00A0-$00A2) and the STOP key indicator at $0091. Uses A, X registers. Real address: $F69B.

$FFEA - UDTIM
Updates Time of Day ($00A0-$00A2) and STOP key indicator ($0091). Uses A, X registers. Real address: $F69B.


---
Additional information can be found by searching:
- "settim" which expands on set TOD (SETTIM $FFDB)
- "rdtim" which expands on read TOD (RDTIM $FFDE)
- "stop" which expands on STOP key state tracked by UDTIM (STOP $FFE1)
