# NMOS 6510 - NO-EFFECT opcodes: certain NOP variants (immediate and memory forms) fetch operands but have no side effects; used to indicate that some opcode encodings simply do nothing aside from consuming cycles/fetching data.


Function
Halt the CPU. The buses will be set to $FF.
