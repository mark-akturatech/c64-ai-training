# ca65 Users Guide - .PARAMCOUNT: a macro-only pseudo variable replaced by the actual number of parameters passed to the macro invocation. Example shows a macro checking .paramcount and emitting .error on mismatch. See the Macros section for more details.

9.5 .PARAMCOUNT
This builtin pseudo variable is only available in macros. It is replaced by the actual number of parameters that were given in the macro invocation.

Example:


        .macro  foo     arg1, arg2, arg3
        .if     .paramcount <> 3
        .error  "Too few parameters for macro foo"
        .endif
        ...
        .endmacro
  
See section Macros.
