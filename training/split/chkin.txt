# Commodore 64 KERNAL Functions Reference - CHKIN ($FFC6). Sets the specified logical file as the default input device (requires file OPEN). Input: X = logical file number. Uses A, X registers. Real address: indirect via ($031E) then $F20E.

$FFC6 - CHKIN
Sets file as default input (requires OPEN). Input: X = logical number. Uses A, X registers. Real address: ($031E), $F20E.
