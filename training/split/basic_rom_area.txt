# Commodore 64 Complete Memory Map - Source: sta.c64.org - BASIC ROM region $A000-$BFFF: 8 KB BASIC interpreter ROM or RAM if banked out/overlaid. Visibility controlled by processor port $0001 bits #0-#2.

BASIC ROM ($A000-$BFFF)
------------------------

$A000-$BFFF  BASIC ROM/RAM      BASIC interpreter or RAM (8192 bytes)

Visibility controlled by processor port $0001 bits #0-#2.
