# Commodore 64 KERNAL Functions Reference - MEMBOT ($FF9C). Manages BASIC workspace lower boundary. Input: Carry (0 = restore, 1 = save); if Carry=0 then X/Y = address to restore; if Carry=1 then X/Y returned = saved address. Uses X, Y registers. Real address: $FE34.

$FF9C - MEMBOT
Manages BASIC workspace lower boundary. Input: Carry (0=restore, 1=save); X/Y = address if Carry=0. Output: X/Y = address if Carry=1. Uses X, Y registers. Real address: $FE34.
