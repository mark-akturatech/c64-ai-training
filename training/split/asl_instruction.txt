# 6502 Instructions summary - ASL (Arithmetic Shift Left). Addressing modes and opcodes: Absolute ASL $aaaa ($0E, 3 bytes) affects N,Z,C ; Zero Page ASL $aa ($06, 2) ; Accumulator ASL A ($0A, 1) ; Absolute Indexed X ASL $aaaa,X ($1E, 3) ; Zero Page Indexed X ASL $aa,X ($16, 2). Flags affected: Negative, Zero, Carry.

ASL	Arithmetic Shift 	Absolute	ASL $aaaa	$0E	3	N,Z,C
	Left	Zero Page	ASL $aa	$06	2	
		Accumulator	ASL A	$0A	1	
		Absolute Indexed, X	ASL $aaaa,X	$1E	3	
		Zero Page Indexed, X	ASL $aa,X	$16	2
