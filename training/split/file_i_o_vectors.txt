# Commodore 64 Complete Memory Map - Source: sta.c64.org - BASIC I/O and channel vectors $031A-$032A: OPEN/CLOSE/CHKIN/CHKOUT/CLRCHN/CHRIN/CHROUT/STOP/GETIN and related I/O function pointers used by the KERNAL/BASIC I/O layer.

$031A-$031B  OPEN               File open routine (default: $F34A)
$031C-$031D  CLOSE              File close routine (default: $F291)
$031E-$031F  CHKIN              Input file definition routine (default: $F20E)
$0320-$0321  CHKOUT             Output file definition routine (default: $F250)
$0322-$0323  CLRCHN             I/O initialization routine (default: $F333)
$0324-$0325  CHRIN              Data input routine (default: $F157)
$0326-$0327  CHROUT             Data output routine (default: $F1CA)
$0328-$0329  STOP               Stop key check routine (default: $F6ED)
$032A-$032B  GETIN              General input routine (default: $F13E)
$032C-$032D  CLALL              Clear all files routine (default: $F32F)
