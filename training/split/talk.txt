# Commodore 64 KERNAL Functions Reference - TALK ($FFB4). Sends a TALK command to a device on the serial IEC bus. Input: A = device number. Uses A register. Real address: $ED09.

$FFB4 - TALK
Sends TALK command to serial bus. Input: A = device number. Uses A register. Real address: $ED09.
