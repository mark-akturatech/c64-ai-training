# Commodore 64 Complete Memory Map - Source: sta.c64.org - Zero Page $0066-$0072: FAC sign ($0066), polynomial degree counter ($0067), temporary area ($0068), ARG register ($0069-$006D) and ARG sign ($006E), plus pointers used for string comparison ($006F-$0070) and array/polynomial tables ($0071-$0072).

$0066   FAC Sign                Sign bit for FAC
$0067   Polynomial Deg          Polynomial degree counter
$0068   Temp Area               General temporary storage
$0069-$006D  ARG                Second arithmetic register (5 bytes)
$006E   ARG Sign                Sign bit for ARG
$006F-$0070  String Compare     Pointer to first string in comparison
$0071-$0072  Array/Poly Ptr     Array pointer or polynomial table pointer
