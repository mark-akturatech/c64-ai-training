# COMMODORE 64 - RTI — Return from interrupt. Operation: Pull processor status P from stack and PC from stack. Flags and PC restored from stack. Reference (9.6). Addressing mode: Implied. Opcode 40, size 1 byte, 6 cycles (note: opcode shown as 4D in source but documented behavior is RTI).
