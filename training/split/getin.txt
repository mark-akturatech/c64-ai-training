# Commodore 64 KERNAL Functions Reference - GETIN ($FFE4). Reads a byte from the default input. Output: A = byte read. Uses A, X, Y registers. Real address: indirect via ($032A) then $F13E.

$FFE4 - GETIN
Reads byte from default input. Output: A = byte. Uses A, X, Y registers. Real address: ($032A), $F13E.
