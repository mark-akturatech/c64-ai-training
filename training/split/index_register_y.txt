# Architecture of 6502 Microprocessor - Index Register Y (8 bits). Typically used as a counter or memory offset; supports compare, increment, and decrement operations similar to X, but without the special interaction with the stack pointer.


Index Register Y	8 bits		Y

Usually used to hold counters or offsets for accessing memory.  Contents can be compared with memory locations and incremented or decremented.
