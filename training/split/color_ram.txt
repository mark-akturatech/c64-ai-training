# Commodore 64 Complete Memory Map - Source: sta.c64.org - Color RAM $D800-$DBFF: screen color memory (1000 bytes) stored at $D800-$DBE7 (bits 0-3 only) and a small unused region $DBE8-$DBFF (24 bytes). Color RAM is separate from screen RAM and controls character colors.

Color RAM ($D800-$DBFF)
------------------------

$D800-$DBE7  Color Memory       Screen color data (1000 bytes, bits 0-3 only)
$DBE8-$DBFF  Unused             (24 bytes)
