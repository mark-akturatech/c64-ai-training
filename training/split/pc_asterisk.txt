# ca65 Users Guide - The '*' pseudo variable: reading returns the program counter (PC) at the start of the current input line. Assignment to '*' is only possible when .FEATURE pc_assignment is enabled; nevertheless .ORG is the recommended way to change the location counter.

9.1 *
Reading this pseudo variable will return the program counter at the start of the current input line.

Assignment to this variable is possible when .FEATURE pc_assignment is used. Note: You should not use assignments to *, use .ORG instead.
