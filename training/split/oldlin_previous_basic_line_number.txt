# ********* - OLDLIN ($3B-$3C) stores the previous (last executed) BASIC line number when program execution ends; CONT restores this value into CURLIN.

59-60         $3B-$3C        OLDLIN
Previous BASIC Line Number

When program execution ends, the last line number executed is stored
here, and restored to location 57 ($39) by CONT.


---
Additional information can be found by searching:
- "curlin_current_basic_line_number" which expands on CURLIN and OLDLIN are used together for STOP/CONT/BREAK behavior
