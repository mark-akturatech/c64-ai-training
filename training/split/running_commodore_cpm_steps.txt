# COMMODORE 64 - Simple step-by-step instructions to run the Commodore CP/M provided with the Z-80 cartridge: (1) LOAD the CP/M program from disk, (2) type RUN, (3) press RETURN.

  RUNNING COMMODORE CP/M(R)

    To run CP/M(R):

      1) LOAD the CP/M(R) program from your disk drive.
      2) Type RUN.
      3) Hit the <RETURN> key.
