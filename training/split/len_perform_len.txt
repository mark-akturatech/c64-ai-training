# ********* - LEN: Performs the LEN function by reading the length field from a string descriptor and converting that integer length into a floating-point value for RETURN.


46972         $B77C          LEN
Perform LEN

The LEN function is performed by obtaining the string length from the
descriptor and converting it to a floating point number.
