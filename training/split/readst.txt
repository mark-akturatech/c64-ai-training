# Commodore 64 KERNAL Functions Reference - READST ($FFB7). Retrieves device status / ST variable (also clears RS232 status). Output: A = status. Uses A register. Real address: $FE07.

$FFB7 - READST
Retrieves device status/ST variable (RS232 status cleared). Output: A = status. Uses A register. Real address: $FE07.
