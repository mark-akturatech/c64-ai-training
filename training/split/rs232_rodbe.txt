# - C64 RAM Map (Mapping The Commodore 64) - $029E RODBE: RS-232 index to the end of the transmit buffer. Points to the ending byte within the 256-byte transmit buffer and is used to add data to that buffer.

$029E        RODBE   RS-232: Index to End of Transmit Buffer
                     
                     This index points to the ending byte within the 256-byte RS-232
                     transmit buffer, and is used to add data to that buffer.
                     

---

- "rs232_rodbs" which expands on start index of transmit buffer
