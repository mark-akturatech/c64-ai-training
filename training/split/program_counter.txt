# Architecture of 6502 Microprocessor - Program Counter (PC), a 16-bit register that contains the address of the next instruction. The hardware increments it automatically; control flow instructions (jumps, branches, subroutines, returns) alter it.


Program Counter	16 bits		PC


Contains the address of the next instruction to be executed.  Automatically incremented by the hardware, but can be altered by a jump, branch or subroutine call / return.
