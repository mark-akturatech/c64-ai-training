# Datassette Encoding - Checksum algorithm used for a 192-byte payload: initialize checksum = $00, then sequentially XOR each payload byte into checksum. Formula: checksum = $00 XOR byte1 XOR byte2 XOR ... XOR byte192. This helps detect tape dropouts and corruption.

Checksum Calculation
--------------------

The checksum uses sequential XOR operations starting with $00, protecting
against data corruption from tape dropouts:

    checksum = $00 XOR byte1 XOR byte2 XOR ... XOR byte192
