Table 4.6. Math Functions
Function

Description

abs(x)

atan(x)

atan2(y,x)

Returns the angle of the coordinate (x,y) relative to the

cbrt(x)

Returns the cube root of x.

ceil(x)

Rounds up to the nearest integer.

cos(r)

Returns the cosine of r.

cosh(x)

Returns the hyperbolic cosine of r.

exp(x)

Returns ex.

expm1(x)

Returns ex-1.
