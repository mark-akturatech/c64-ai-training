# - C64 RAM Map (Mapping The Commodore 64) - $029C RIDBS: RS-232 index to the start of the receive buffer. Points to the starting byte within the 256-byte receive buffer and is used to remove data from that buffer.

$029C        RIDBS   RS-232: Index to Start of Receive Buffer
                     
                     This index points to the starting byte within the 256-byte RS-232
                     receive buffer, and is used to remove data from that buffer.
