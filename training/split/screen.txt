# Commodore 64 KERNAL Functions Reference - SCREEN ($FFED). Returns screen dimensions. Output: X = 40 columns; Y = 25 rows. Uses X, Y registers. Real address: $E505.

$FFED - SCREEN
Returns screen dimensions. Output: X = 40 columns; Y = 25 rows. Uses X, Y registers. Real address: $E505.
