# MACHINE - Opcode encoding grid for shift/rotate and register/memory modify instructions (ASL, ROL, LSR, ROR, STX, LDX, DEC, INC). Lists hex opcodes by addressing mode (IMM, ZP, ZP,X, ZP,Y, ABS, ABS,X, ABS,Y) and notes that these opcodes end in -2, -6, or -E.

            *----------------------------------------------------*
            |       IMM   ZP   ZP,X   ZP,Y   ABS   ABS,X   ABS,Y |
            |        2     2     2      2     3      3       3   |
            |     +----------------------------------------------+
            | ASL |       06    16           0E     1E           |
            | ROL |       26    36           2E     3E           |
            | LSR |       46    56           4E     5E           |
            | ROR |       66    76           6E     7E           |
            | STX |       86           96    8E                  |
            | LDX | A2    A6           B6    AE             BE   |
            | DEC |       C6    D6           CE     DE           |
            | INC |       E6    F6           EE     FE           |
            *-----+----------------------------------------------*
                        Op Code ends in -2, -6, or -E
