# ********* - Documents EAL ($AE-$AF), the pointer to the ending address for SAVE/LOAD/VERIFY operations. Set by the Kernal SAVE routine to mark the end of the region to be saved or loaded.

174-175       $AE-$AF        EAL
Pointer to Ending Address of Load (End of Program)

This location is set by the Kernal routine SAVE to point to the ending
address for SAVE, LOAD, or VERIFY.
