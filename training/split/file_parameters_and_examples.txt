# Kick Assembler - Shows adding files from different sources to a disk with .disk and a list of file parameter maps. Examples include prg files from segments, prg from prgFiles on disk, sid files, hidden files (hide), locked files (type="prg<"). Also shows .segment definitions (BORDER_COLORS, BACK_COLORS, HIDDEN) used as intermediate segments providing file contents.

If set to true, the file will
not be shown in the directory.
