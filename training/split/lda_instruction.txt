# 6502 Instructions summary - LDA (Load Accumulator). Addressing modes and opcodes: Absolute LDA $aaaa ($AD, 3 bytes) sets N,Z; Zero Page LDA $aa ($A5, 2) ; Immediate LDA #$aa ($A9, 2) ; Absolute Indexed X LDA $aaaa,X ($BD, 3) ; Absolute Indexed Y LDA $aaaa,Y ($B9, 3) ; Zero Page Indexed X LDA $aa,X ($B5, 2) ; Indexed Indirect LDA ($aa,X) ($A1, 2) ; Indirect Indexed LDA ($aa),Y ($B1, 2). Flags affected: Negative, Zero.

LDA	Load Accumulator	Absolute	LDA $aaaa	$AD	3	N,Z
		Zero Page	LDA $aa	$A5	2	
		Immediate	LDA #$aa	$A9	2	
		Absolute Indexed, X	LDA $aaaa,X	$BD	3	
		Absolute Indexed, Y	LDA $aaaa,Y	$B9	3	
		Zero Page Indexed, X	LDA $aa,X	$B5	2	
		Indexed Indirect	LDA ($aa,X)	$A1	2	
		Indirect Indexed	LDA ($aa),Y	$B1	2
