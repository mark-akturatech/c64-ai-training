# C64 PETSCII and Screen Codes Reference - sta.c64.org - Special-case PETSCII code $FF (decimal 255): documented as the BASIC token for the pi symbol. Describes internal conversion behavior: $FF is converted to $DE when printed; $DE is converted back to $FF when fetched from the screen; when reading the keyboard buffer $DE is returned for Shift-up arrow (no conversion performed at that stage).

--------------------------------------------------------------------------------
PETSCII CODE $FF (SPECIAL CASE)
--------------------------------------------------------------------------------

255  $FF   Pi symbol (BASIC token)
             Internally converted to $DE when printed.
             $DE is converted back to $FF when fetched from screen.
             When reading keyboard buffer, $DE is returned for Shift-up arrow
             (no conversion takes place at that stage).
