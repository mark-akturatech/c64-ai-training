# Commodore 64 Complete Memory Map - Source: sta.c64.org - Zero Page $00BD-$00C5: RS232 parity/aux buffering ($00BD), datasette block counter ($00BE), unused ($00BF), datasette motor switch ($00C0), save start addresses ($00C1-$00C2) and secondary start ($00C3-$00C4), plus previous key matrix entry ($00C5).

$00BD   RS232 Parity            RS232 parity or datasette buffer
$00BE   Block Counter           Datasette block counter
$00BF   Unused
$00C0   Motor Switch            Datasette motor control switch
$00C1-$00C2  Save Start         Start address for save operations
$00C3-$00C4  Secondary Start    Start address for secondary address 0
$00C5   Prev Key Matrix         Previous key pressed (matrix code)
