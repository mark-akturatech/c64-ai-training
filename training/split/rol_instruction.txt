# 6502 Instructions summary - ROL (Rotate Left). Addressing modes and opcodes: Absolute ROL $aaaa ($2E, 3 bytes) affects N,Z,C ; Zero Page ROL $aa ($26, 2) ; Accumulator ROL A ($2A, 1) ; Absolute Indexed X ROL $aaaa,X ($3E, 3) ; Zero Page Indexed X ROL $aa,X ($36, 2). Flags affected: Negative, Zero, Carry.

		Zero Page	ROL $aa	$26	2	
		Accumulator	ROL A	$2A	1
