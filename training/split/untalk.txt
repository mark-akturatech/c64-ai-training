# Commodore 64 KERNAL Functions Reference - UNTALK ($FFAB). Sends the UNTALK command on the serial IEC bus to end a TALK session. Uses A register. Real address: $EDEF.

$FFAB - UNTALK
Sends UNTALK command to serial bus. Uses A register. Real address: $EDEF.
