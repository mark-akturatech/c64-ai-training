# ********* - BITS ($68): overflow byte for FAC1. Used as an intermediate storage location during conversions (integer or text string to floating point) to hold overflowed least-significant digits.


104           $68            BITS
Floating Point Accumulator #1: Overflow Digit

This location contains the overflow byte.  The overflow byte is used
in an intermediate step of conversion from an integer or text string
to a floating point number.
