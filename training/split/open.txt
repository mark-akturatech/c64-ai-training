# Commodore 64 KERNAL Functions Reference - OPEN ($FFC0). Opens a file (requires prior SETLFS and SETNAM). Uses registers A, X, Y. Real address: indirect via ($031A) then $F34A.

$FFC0 - OPEN
Opens file (requires SETLFS/SETNAM). Uses A, X, Y registers. Real address: ($031A), $F34A.


---
Additional information can be found by searching:
- "setlfs" which expands on SETLFS configures logical number, device, secondary address
- "setnam" which expands on SETNAM sets filename and length
- "close" which expands on CLOSE closes files opened by OPEN
