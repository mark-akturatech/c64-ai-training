# Commodore 64 KERNAL Functions Reference - SCINIT ($FF81). Initializes VIC chip, restores keyboard/screen I/O, clears display, sets PAL/NTSC mode and interrupt timer. No input/output parameters. Uses registers A, X, Y. Real address: $FF5B.

$FF81 - SCINIT
Initializes VIC chip, restores keyboard/screen I/O, clears display, sets PAL/NTSC mode and interrupt timer. No input/output parameters. Uses A, X, Y registers. Real address: $FF5B.
