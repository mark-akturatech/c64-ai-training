# ********* - Definition of JMPER ($54-$56): contains a 6502 JMP instruction ($4C) followed by the address of a required BASIC function taken from the function table at 41042 ($A052).

84-86         $54-$56        JMPER
Jump to Function Instruction

The first byte is the 6502 JMP instruction ($4C), followed by the
address of the required function taken from the table at 41042
($A052).
