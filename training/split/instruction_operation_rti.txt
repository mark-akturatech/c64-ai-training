# 6502 Microprocessor - RTI pseudocode: pull status from stack (SET_SR), then pull return address (low/high) and restore PC.

/* RTI */
    src = PULL();
    SET_SR(src);
    src = PULL();
    src |= (PULL() << 8);	/* Load return address from stack. */
    PC = (src);
