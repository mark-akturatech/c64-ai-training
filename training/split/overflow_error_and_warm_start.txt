# - Fully Commented Commodore 64 ROM Disassembly (English) - Overflow error handler entry used when arithmetic overflow occurs: sets the error code to $0F (overflow error) and jumps to the central error/warm-start routine to report the error and restart.


                                *** do overflow error then warm start
.,B97E A2 0F    LDX #$0F        error $0F, overflow error
.,B980 4C 37 A4 JMP $A437       do error #X then warm start
