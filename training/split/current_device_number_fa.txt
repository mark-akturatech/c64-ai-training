# ********* - FA ($BA) — Holds the number of the device currently being used. Lists device-number assignments: 0 = keyboard, 1 = Datasette recorder, 2 = RS-232/User Port, 3 = screen, 4–5 = printer, 8–11 = disk.

186           $BA            FA
Current Device Number

This location holds the number of the device that is currently being
used.  Device number assignments are as follows:

0    = Keyboard
1    = Datasette Recorder
2    = RS-232/User Port
3    = Screen
4-5  = Printer
8-11 = Disk


---
Additional information can be found by searching:
- "current_secondary_address_sa" which expands on secondary addresses are interpreted differently depending on the device number
- "current_logical_file_number_la" which expands on logical file numbers identify open files for the currently selected device
