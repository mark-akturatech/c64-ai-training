# - Fully Commented Commodore 64 ROM Disassembly (English) - Flag block done and exit interrupt: SEC, ROR $B6 (mark buffer high byte negative and flag sync/data/checksum bytes written), BMI to restore/exit (branch always). Prepares for exit of tape write operation.

                                *** flag block done and exit interrupt
.,FBC8 38       SEC             set carry flag
.,FBC9 66 B6    ROR $B6         set buffer address high byte negative, flag all sync,
                                data and checksum bytes written
.,FBCB 30 3C    BMI $FC09       restore registers and exit interrupt, branch always
