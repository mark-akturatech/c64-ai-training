# Commodore 64 KERNAL Functions Reference - CHROUT ($FFD2). Writes a byte to the default output. Input: A = byte to write. Real address: indirect via ($0326) then $F1CA.

$FFD2 - CHROUT
Writes byte to default output. Input: A = byte. Real address: ($0326), $F1CA.
