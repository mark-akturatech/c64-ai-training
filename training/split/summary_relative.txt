# 6502 ADDRESSING MODES - SUMMARY: Relative addressing recap for branches: operand is a two's complement offset (-128 to +127) added to the address after the branch to produce the target. Example: 4545 BEQ $13 will branch to $4547 + $13 = $455A if zero flag is set. Example following LDA at $4547.
