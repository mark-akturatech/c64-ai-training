# MACHINE - B-Series Tri Port Adapter 6525 (Tri Port 2) register usage ($DF00-$DF07). Mappings include keyboard matrix input ($DF00), select lines ($DF01), CRT mode and keyboard read controls ($DF02), and corresponding data direction registers ($DF03-$DF05), with notes on unused registers.

      +-------+-------+-------+-------+-------+-------+-------+-------+

$DF01 |                            Select                             | 57089
      +-------+-------+-------+-------+-------+-------+-------+-------+
$DF02 |   CRT Mode    |                 Keyboard Read                 | 57090
      +-------+-------+-------+-------+-------+-------+-------+-------+
$DF03 |            Data Direction Register for $DF00 (out)            | 57091
      +-------+-------+-------+-------+-------+-------+-------+-------+
$DF04 |            Data Direction Register for $DF01 (out)            | 57092

      +-------+-------+-------+-------+-------+-------+-------+-------+
