# Commodore 64 Complete Memory Map - Source: sta.c64.org - CIA #1 timer control $DC0E-$DC0F (Timer A/B control registers) and CIA#1 image mirror area $DC10-$DCFF (register images repeated every $10 bytes).

Timer Control:

$DC0E   Timer A Control         Start, output mode, and count source
$DC0F   Timer B Control         Start, output mode, and count source

Mirror Area:

$DC10-$DCFF  CIA#1 Images       Register images (repeated every $10 bytes)
