# Commodore IEC Serial Bus Protocol Reference - pagetable.com - REL (relative) files: random access using fixed record sizes (1-254 bytes). POSITION command sets record pointers using binary-encoded arguments.

Relative Files (REL):
  Enable random access using fixed record sizes (1-254 bytes).
  The POSITION command sets record pointers using binary-encoded arguments.
