# An Introduction to Programming C-64 Demos - Zero page performance tip: LDA/STA to zero page addresses take fewer cycles (3 cycles) than absolute addressing (4 cycles); use zero page for frequently accessed variables.

Use the Zero Page
An LDA or STA to a normal memory address takes 4 clock cycles, but if you use an address in the zero page, they only take 3 cycles. Need I say more? Maybe, so here goes: use the zero page!
