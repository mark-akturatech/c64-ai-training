# - Fully Commented Commodore 64 ROM Disassembly (English) - Small helper that forces the system 'top of memory' high byte to $F0 and returns. (Performs SEC; LDA #$F0; JMP to vector that finalizes top-of-memory.) Used when RS232 buffer allocation needs the F000 page.

                                *** set the top of memory to F0xx
.,F47D 38       SEC             read the top of memory
.,F47E A9 F0    LDA #$F0        set $F000
.,F480 4C 2D FE JMP $FE2D       set the top of memory and return
