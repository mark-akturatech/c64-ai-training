# ********* - C3PO ($94): flag indicating the serial output routines have buffered a character for transmission (output buffer is occupied).

148           $94            C3PO
Flag: Serial Bus--Output Character Was Buffered

This location is used by the serial output routines to indicate that a
character has been placed in the output buffer and is waiting to be
sent.


---
Additional information can be found by searching:
- "bsour_serial_buffered_character_0x95" which expands on holds the actual buffered character for serial output
- "st_status_io_status_word_0x90" which expands on serial device status bits and timeouts
