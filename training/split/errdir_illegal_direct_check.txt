# ********* - ERRDIR: Check for illegal direct mode execution (routine at $B3A6). Describes how this routine is called by statements that prohibit execution in direct mode, checks a flag set when a line without a line number is entered, and causes an ILLEGAL DIRECT error if the flag indicates direct mode.

45990         $B3A6          ERRDIR
Check If the Program is Running in Direct Mode, and If So Issue an Error

This routine is called by statements that prohibit execution in direct
mode.  It checks a flag that is set when a line without a linenumber
is entered, and causes an ILLEGAL DIRECT error if the flag is set.
