# Commodore 64 KERNAL Functions Reference - SETTIM ($FFDB). Sets the Time-of-Day (TOD) clock at $00A0-$00A2. Input: A/X/Y = new TOD value. Real address: $F6E4.

$FFDB - SETTIM
Sets Time of Day at $00A0-$00A2. Input: A/X/Y = new TOD value. Real address: $F6E4.


---
Additional information can be found by searching:
- "rdtim" which expands on read current TOD (RDTIM $FFDE)
- "udtim" which expands on update TOD and STOP indicator (UDTIM $FFEA)
