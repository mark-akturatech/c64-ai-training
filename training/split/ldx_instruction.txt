# 6502 Instructions summary - LDX (Load X Register). Addressing modes and opcodes: Absolute LDX $aaaa ($AE, 3 bytes) sets N,Z; Zero Page LDX $aa ($A6, 2) ; Immediate LDX #$aa ($A2, 2) ; Absolute Indexed Y LDX $aaaa,Y ($BE, 3) ; Zero Page Indexed Y LDX $aa,Y ($B6, 2). Flags affected: Negative, Zero.

LDX	Load X Register	Absolute	LDX $aaaa	$AE	3	N,Z
		Zero Page	LDX $aa	$A6	2	
		Immediate	LDX #$aa	$A2	2	
		Absolute Indexed, Y	LDX $aaaa,Y	$BE	3	
		Zero Page Indexed, Y	LDX $aa,Y	$B6	2
