# ca65 Users Guide - Short command-line options (-D, -I, -U, -V, -W, -d, -g, -x, -h, -i, -l, -mm, -o, -s, -S, -t, -v) with brief one-line descriptions.

Short options:
  -D name[=value]               Define a symbol
  -I dir                        Set an include directory search path
  -U                            Mark unresolved symbols as import
  -V                            Print the assembler version
  -W n                          Set warning level n
  -d                            Debug mode
  -g                            Add debug info to object file
  -x                            Expand macros in the listing
                                repeat -x for full expansion
  -h                            Help (this text)
  -i                            Ignore case of symbols
  -l name                       Create a listing file if assembly was ok
  -mm model                     Set the memory model
  -o name                       Name the output file
  -s                            Enable smart mode
  -S                            Generate segment offsets in listing
  -t sys                        Set the target system
  -v                            Increase verbosity
