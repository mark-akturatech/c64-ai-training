# 6502 Instructions summary - LSR (Logical Shift Right). Addressing modes and opcodes: Absolute LSR $aaaa ($4E, 3 bytes) affects N,Z,C ; Zero Page LSR $aa ($46, 2) ; Accumulator LSR A ($4A, 1) ; Absolute Indexed X LSR $aaaa,X ($5E, 3) ; Zero Page Indexed X LSR $aa,X ($56, 2). Flags affected: Negative, Zero, Carry.
