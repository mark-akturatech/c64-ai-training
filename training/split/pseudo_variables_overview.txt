# ca65 Users Guide - Introduction to assembler pseudo variables: they are readable in all cases and in some special cases writable. General section header for the following specific pseudo variables.

9. Pseudo variables
Pseudo variables are readable in all cases, and in some special cases also writable.


---
Additional information can be found by searching:
- "pc_asterisk" which expands on Program counter pseudo variable (*)
- "asize_pseudo_variable" which expands on Accumulator size (.ASIZE)
- "cpu_pseudo_variable" which expands on CPU identification (.CPU)
- "isize_pseudo_variable" which expands on Index size (.ISIZE)
- "paramcount_pseudo_variable" which expands on Macro parameter count (.PARAMCOUNT)
- "time_pseudo_variable" which expands on Translation time (.TIME)
- "version_pseudo_variable" which expands on Assembler version (.VERSION)
