# MACHINE - PET/CBM example to change start-of-variables pointer: display SOV pointer with '.M 002A 002B' and change pointer bytes (example '.:002D C8 04 .. .. .. .. .. ..').

To change the start-of-variables pointer to a location above the machine
language program, display the SOV pointer with .M 002A 002B and change the
pointer to

  .:002D C8 04 .. .. .. .. .. ..
