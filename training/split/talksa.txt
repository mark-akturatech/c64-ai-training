# Commodore 64 KERNAL Functions Reference - TALKSA ($FF96). Transmits a TALK secondary address via the serial IEC bus (requires prior TALK call). Input: A = secondary address. Uses A register. Real address: $EDC7.

$FF96 - TALKSA
Transmits TALK secondary address via serial bus (requires prior TALK call). Input: A = secondary address. Uses A register. Real address: $EDC7.
