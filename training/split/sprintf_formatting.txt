# ca65 Users Guide - Documentation for the .SPRINTF builtin function: printf-style formatting that returns a string. It expects a format string as the first argument; additional arguments depend on the format. Notes missing features compared to C printf (length modifiers, variable width). Provides an example generating an identifier using .sprintf: .ident (.sprintf ("%s%03d", "label", num)): where num = 3.

10.24 .SPRINTF
Builtin function. It expects a format string as first argument. The number and type of the following arguments depend on the format string. The format string is similar to the one of the C printf function. Missing things are: Length modifiers, variable width.

The result of the function is a string.

Example:


        num     = 3

        ; Generate an identifier:
        .ident (.sprintf ("%s%03d", "label", num)):
