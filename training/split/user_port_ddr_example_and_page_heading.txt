# COMMODORE 64 - Assembly example showing how to set the Data Direction Register (DDR) of the user port to 0 (LDA #0; STA (POINT),Y). Includes the page heading/footer marker for the 'BASIC TO MACHINE LANGUAGE' section.

    LDA #0          ;OFFSET FOR DDR OF THE USER PORT
    STA (POINT),Y   ;SET DDR TO 0
