# Commodore 64 Complete Memory Map - Source: sta.c64.org - System Area $028B-$0297: key repeat delays ($028B-$028D), shift key status and previous shift state ($028D-$028E), shift routine address ($028F-$0290), commodore-shift enable ($0291), scroll direction ($0292), RS232 control/command and RS232 out timer defaults ($0293-$0296), RS232 status ($0297).

$028C   Repeat Start Delay      Initial delay before first repeat
$028D   Shift Key Status        Current shift key state indicators
$028E   Prev Shift Status       Previous shift key state
$028F-$0290  Shift Routine Addr Shift routine execution address (default: $EB48)
$0291   Commodore-Shift         Commodore-Shift enable flag
$0292   Scroll Direction        Direction of screen scroll
$0293   RS232 Control           RS232 baud rate and byte configuration
$0294   RS232 Command           RS232 synchronization and parity control
$0295-$0296  RS232 Out Timer    RS232 output timer default value
$0297   RS232 Status            RS232 device status for input/output
