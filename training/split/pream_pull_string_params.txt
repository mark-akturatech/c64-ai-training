# ********* - PREAM: Pulls the string-function parameters from the stack for LEFT$, RIGHT$, and MID$. Used to obtain the first two parameters common to these commands.


46945         $B761          PREAM
Pull String Function Parameters from Stack for LEFT$, RIGHT$, and MID$

This routine is used to obtain the first two parameters for all three
of these commands.
