# Commodore IEC Serial Bus Protocol Reference - pagetable.com - Burst API: low-level disk access via U0 subcommands supporting physical sector operations and file transfers using alternate serial protocols. Available on 1571 and 1581 drives with the C128.

Burst API:
  Provides low-level disk access via U0 subcommands, supporting physical
  sector operations and file transfers using alternate serial protocols.
  Available on 1571 and 1581 drives with the C128.
