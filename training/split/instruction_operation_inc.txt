# 6502 Microprocessor - INC pseudocode: increment memory value by one (wrap to 8-bit), set Sign and Zero, store back.

/* INC */
    src = (src + 1) & 0xff;
    SET_SIGN(src);
    SET_ZERO(src);
    STORE(address, (src));
