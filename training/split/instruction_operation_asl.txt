# 6502 Microprocessor - ASL pseudocode: shift left, top bit -> Carry, low bit filled with 0; set sign and zero; store result to memory or accumulator depending on mode.

/* ASL */
    SET_CARRY(src & 0x80);
    src <<= 1;
    src &= 0xff;
    SET_SIGN(src);
    SET_ZERO(src);
    STORE src in memory or accumulator depending on addressing mode.
