Directory Listing:
  The dollar-sign syntax "$" retrieves directory contents, supporting
  wildcards and filtering by file type or timestamp when available.

  Examples:
    "$"        - Full directory listing
    "$0:*"     - All files on drive 0
    "$0:*.PRG" - Only PRG files
