# 6502 Microprocessor - LSR pseudocode: set Carry from bit0, shift right by one, set Sign and Zero, store into memory or accumulator depending on mode.

/* LSR */
    SET_CARRY(src & 0x01);
    src >>= 1;
    SET_SIGN(src);
    SET_ZERO(src);
    STORE src in memory or accumulator depending on addressing mode.
