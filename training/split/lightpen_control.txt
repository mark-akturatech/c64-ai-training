# COMMODORE 64 - LIGHTPEN CONTROL entry explaining that lightpen applications can be performed with any lightpen that fits the C64 game port connector.

  LIGHTPEN CONTROL        Applications requiring the use of a lightpen
                          can be performed by any lightpen that will fit
                          the COMMODORE 64 game port connector.
