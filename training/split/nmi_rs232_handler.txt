# ********* - NMI RS-232 handler (continuation): checks if it's time to send or receive a bit on the RS-232 channel and performs the appropriate action when the RS-232 timer indicates a bit period.

65138         $FE72
NMI RS-232 Handler

This is the part of the NMI handler that checks if it is time to
receive or send a bit on the RS-232 channel, and takes the appropriate
action if it is indeed the time.
