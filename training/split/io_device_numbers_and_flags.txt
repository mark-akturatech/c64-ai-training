# Commodore 64 Complete Memory Map - Source: sta.c64.org - Zero Page $0099-$009F: input and output device numbers ($0099-$009A), datasette parity/byte-ready flags ($009B-$009C), error display and RS232/datasette output/error counter ($009D-$009E), and auxiliary datasette counter ($009F).

$0099   Input Device            Current input device (default: $00)
$009A   Output Device           Current output device (default: $03)
$009B   Parity Bit              Datasette parity indicator
$009C   Byte Ready              Datasette byte ready flag
$009D   Error Display           System message display control
$009E   Output Buffer           RS232/datasette output byte or error counter
$009F   Aux Counter             Datasette auxiliary counter
