# ********* - READY kernel routine: prints the word READY, sets the Kernal message flag to indicate direct mode is active, and falls through to the main BASIC loop.

42089-42099   $A474-$A47F    READY
Print READY

This routine displays the word READY, sets the Kernal message flag to
show that direct mode is operative, and falls through to the main
BASIC loop.
