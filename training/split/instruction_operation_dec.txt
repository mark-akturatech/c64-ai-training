# 6502 Microprocessor - DEC pseudocode: decrement memory value by one (wrap to 8-bit), set Sign and Zero, store back to memory.

/* DEC */
    src = (src - 1) & 0xff;
    SET_SIGN(src);
    SET_ZERO(src);
    STORE(address, (src));
