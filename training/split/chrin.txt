# Commodore 64 KERNAL Functions Reference - CHRIN ($FFCF). Reads a byte from the default input or a line from the screen. Output: A = byte read. Uses A, Y registers. Real address: indirect via ($0324) then $F157.

$FFCF - CHRIN
Reads from default input or line from screen. Output: A = byte. Uses A, Y registers. Real address: ($0324), $F157.


---
Additional information can be found by searching:
- "chrout" which expands on writing to default output (CHROUT $FFD2)
- "getin" which expands on lower-level input reading (GETIN $FFE4)
