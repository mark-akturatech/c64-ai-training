# - C64 RAM Map (Mapping The Commodore 64) - $0300-$0301 IERROR: vector to the BASIC error message routine (points to ERROR routine at $E38B / 58251).

$0300-$0301  IERROR  Vector to the Print BASIC Error Message Routine
                     
                     This vector points to the address of the ERROR routine at 58251
                     ($E38B).
