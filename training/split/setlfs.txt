# Commodore 64 KERNAL Functions Reference - SETLFS ($FFBA). Configures file parameters: Input: A = logical file number; X = device number; Y = secondary address. Real address: $FE00.

$FFBA - SETLFS
Configures file parameters. Input: A = logical number; X = device number; Y = secondary address. Real address: $FE00.


---
Additional information can be found by searching:
- "setnam" which expands on SETNAM sets the filename for subsequent OPEN
- "open" which expands on OPEN uses parameters from SETLFS/SETNAM
