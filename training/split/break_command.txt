# Architecture of 6502 Microprocessor - Break Command Flag (B). The B bit is set when a BRK instruction executes; BRK generates an interrupt to allow handling of the software break.


Break Command 	B

The break command bit is set when a BRK instruction has been executed and an interrupt has been generated to process it.
