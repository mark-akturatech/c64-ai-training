# - Commented Commodore 64 KERNAL Disassembly (Magnus Nyman) - The string placed in the keyboard buffer when <Shift-RUN> is pressed: the message 'LOAD <CR> RUN <CR>' (bytes for 'LOAD', carriage return, 'RUN', carriage return).

                                *** SHIFT-RUN EQUIVALENT
                                This is the message LOAD <CR> RUN <CR>, which is placed in
                                the keyboard buffer when <shift-RUN> is pressed.
.:ECE7 4C 4F 41 44 0D 52 55 4E  LOAD <CR> RUN <CR>
