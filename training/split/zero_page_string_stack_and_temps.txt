# Commodore 64 Complete Memory Map - Source: sta.c64.org - Zero Page $0016-$0029: string stack and temporary areas. Includes string stack pointer ($0016), previous pointer ($0017-$0018), string stack temp area ($0019-$0021), general temporary storage ($0022-$0025), and auxiliary arithmetic register ($0026-$0029) used for division/multiplication.

$0019-$0021  String Stack       Temporary expression processing area (9 bytes)
$0022-$0025  Temp Area          General-purpose temporary storage (4 bytes)
$0026-$0029  Aux Arithmetic     Division/multiplication register (4 bytes)
