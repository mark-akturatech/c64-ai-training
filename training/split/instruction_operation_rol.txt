# 6502 Microprocessor - ROL pseudocode: shift src left, insert carry into bit0 if set, set Carry from overflow >0xff, mask to 8-bit, update Sign and Zero, store result.

/* ROL */
    src <<= 1;
    if (IF_CARRY()) src |= 0x1;
    SET_CARRY(src > 0xff);
    src &= 0xff;
    SET_SIGN(src);
    SET_ZERO(src);
    STORE src in memory or accumulator depending on addressing mode.
