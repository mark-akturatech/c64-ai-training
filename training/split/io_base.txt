# Commodore 64 KERNAL Functions Reference - IOBASE ($FFF3). Returns the CIA #1 base address. Output: X/Y = $DC00. Uses X, Y registers. Real address: $E500.

$FFF3 - IOBASE
Returns CIA #1 base address. Output: X/Y = $DC00. Uses X, Y registers. Real address: $E500.

Additional information can be found by searching:
- "ioinit" which expands on CIA initialization performed by IOINIT ($FF84)
