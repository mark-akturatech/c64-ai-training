# Commodore 64 KERNAL Functions Reference - SAVE ($FFD8). Saves a file (requires SETLFS/SETNAM). Input: A = zero page register holding start address; X/Y = end address plus 1. Output: Carry flag (0 = success, 1 = error); A = error code if Carry=1. Uses A, X, Y registers. Real address: $F5DD.

$FFD8 - SAVE
Saves file (requires SETLFS/SETNAM). Input: A = zero page register holding start address; X/Y = end address plus 1. Output: Carry (0=success, 1=error); A = error code if Carry=1. Uses A, X, Y registers. Real address: $F5DD.
