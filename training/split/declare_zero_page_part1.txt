# - Fully Commented Commodore 64 KERNAL ROM Disassembly (English, "CBM") - Zero-page and low-memory declarations: defines 6510 I/O ports D6510/R6510, virtual registers for machine language monitor (PCH,PCL,FLGS,ACC,XR,YR,SP), user-modifiable IRQ vectors INVH/INVL, STATUS, device flags STATUS/STKEY/SVXT/VERCK and many temporary variables used by cassette, serial and editor code. Also sets origin to $0000 and uses '*=$0100' marker later. (Contains many zero page symbol definitions and comments.)
