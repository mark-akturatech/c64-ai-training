# COMMODORE 64 - Shows how to enter multi-color display of typed text using POKE 53270 to set multicolor mode and printing a CTRL+3 sequence (PRINT"<CTRL+3>") to set a color. Also notes that COLOR keys can change characters to any color including multicolor.









    By using the <C=> key and the COLOR keys the characters can be changed
  to any color, including multi-color characters. For example, type this
  command:

    POKE 53270,PEEK(53270)OR 16:PRINT"<CTRL+3>";: rem lt.red/ multi-color
  red
