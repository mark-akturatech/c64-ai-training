# Kick Assembler - IParameterMap interface: collection of name -> IValue pairs (used for source code parameters in square brackets). Main methods: exists(), getValue(), getSourceRange(), getParameterNames(). Convenience getters: getBoolValue(name, default), getEnumValue(...), getIntValue(name, default), getStringValue(name, default), IValue getValue(name), ISourceRange getSourceRange(name), Collection<String> getParameterNames().
