# - C64 RAM Map (Mapping The Commodore 64) - $0310: byte containing the 6510 JMP instruction opcode ($4C / 76 decimal) used by the USR command (first byte of JMP instruction placed here).

$0310                Jump Instruction for User Function ($4C)
                     
                     The value here (67, $4C) is first part of the 6510 machine language
                     JuMP instruction for the USR command.
