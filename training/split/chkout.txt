# Commodore 64 KERNAL Functions Reference - CHKOUT ($FFC9). Sets the specified logical file as the default output device (requires file OPEN). Input: X = logical file number. Uses A, X registers. Real address: indirect via ($0320) then $F250.

$FFC9 - CHKOUT
Sets file as default output (requires OPEN). Input: X = logical number. Uses A, X registers. Real address: ($0320), $F250.
