# Commodore 64 Complete Memory Map - Source: sta.c64.org - VIC-II screen control registers $D011-$D018: $D011 (vertical scroll, height/bitmap mode, raster MSB), $D012 raster line (bits 0-7), $D013-$D014 light pen X/Y (read-only), $D015 sprite enable, $D016 screen control 2 (horizontal scroll, width, multicolor), $D017 sprite height (double height), $D018 memory setup (character/bitmap/screen memory pointers).

Screen Control:

$D011   Screen Ctrl 1           Vertical scroll, height, bitmap mode, raster bit 8
$D012   Raster Line             Current/interrupt raster line (bits 0-7)
$D013   Light Pen X             Light pen X-coordinate (bits 1-8, read-only)
$D014   Light Pen Y             Light pen Y-coordinate (read-only)
$D015   Sprite Enable           Enable register for sprites 0-7
$D016   Screen Ctrl 2           Horizontal scroll, width, multicolor mode
$D017   Sprite Height           Double-height stretch register for sprites 0-7
$D018   Memory Setup            Character/bitmap and screen memory pointers
