# Commodore 64 Complete Memory Map - Source: sta.c64.org - Zero Page $00D0-$00F1: EOL switch ($00D0), screen line pointer ($00D1-$00D2), current column/quote mode/line length/current row/input character/insertion count ($00D3-$00D9), and high-byte table for 25 screen line pointers ($00D9-$00F1).

$00CF   Cursor Phase            Cursor visibility phase (0 or 1)
$00D0   EOL Switch              End-of-line indicator
$00D1-$00D2  Screen Line Ptr    Pointer to current screen line
$00D3   Current Column          Cursor column (0-39)
$00D4   Quote Mode              Quotation mode during input
$00D5   Screen Line Len         Screen line length minus 1
$00D6   Current Row             Cursor row (0-24)
$00D7   Input Character         PETSCII code or buffer during I/O
$00D8   Insertions Count        Number of character insertions
$00D9-$00F1  Screen Line Table  High bytes of screen line pointers (25 bytes)
