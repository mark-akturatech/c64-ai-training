# - Fully Commented Commodore 64 ROM Disassembly (English) - Wrapper at $FFED that JMPs to $E505 to return the X,Y organisation of the screen in X and Y registers.

                                *** return X,Y organization of screen
                                this routine returns the x,y organisation of the screen in X,Y
.,FFED 4C 05 E5 JMP $E505       return X,Y organization of screen
