# ********* - VARNAM ($45-$46) stores the current variable name being searched for, in the same two-byte format used in variable descriptors (see variable descriptor description at location 45 ($2D) for format details).

69-70         $45-$46        VARNAM
Current BASIC Variable Name

The current variable name being searched for is stored here, in the
same two- byte format as in the variable value storage area located at
the address pointed to by 45 ($2D).  See that location for an
explanation of the format.


---
Additional information can be found by searching:
- "varpnt_pointer_to_current_variable_descriptor" which expands on VARPNT points to the descriptor (value storage) corresponding to the name in VARNAM
