# Commodore 64 KERNAL Functions Reference - SETNAM ($FFBD). Establishes filename parameters. Input: A = name length; X/Y = pointer to filename in memory. Real address: $FDF9.

$FFBD - SETNAM
Establishes filename parameters. Input: A = name length; X/Y = name pointer. Real address: $FDF9.


---
Additional information can be found by searching:
- "setlfs" which expands on SETLFS configures device and logical file (#) before OPEN
- "open" which expands on OPEN uses filename set by SETNAM
