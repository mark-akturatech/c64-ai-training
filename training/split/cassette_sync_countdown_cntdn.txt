# ********* - Documents CNTDN ($A5), the cassette synchronization-character countdown used to count how many sync characters are sent before the actual data in a tape block.

165           $A5            CNTDN
Cassette Synchronization Character Countdown

Used to count down the number of synchronization characters that are
sent before the actual data in a tape block.
