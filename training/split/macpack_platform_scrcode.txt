# ca65 Users Guide - .MACPACK platform scrcode macros: apple2, atari, and cbm macpacks each define a scrcode macro that takes a string and places it into memory translated into that platform's screen codes.

13.3 .MACPACK apple2
This macro package defines a macro named scrcode. It takes a string as argument and places this string into memory translated into screen codes.

13.4 .MACPACK atari
This macro package defines a macro named scrcode. It takes a string as argument and places this string into memory translated into screen codes.

13.5 .MACPACK cbm
This macro package defines a macro named scrcode. It takes a string as argument and places this string into memory translated into screen codes.
