# Commodore 64 Complete Memory Map - Source: sta.c64.org - CIA #1 registers $DC00-$DC03 port control: $DC00 Port A (keyboard matrix & joystick #2), $DC01 Port B (keyboard rows & joystick #1), $DC02 Port A DDR, $DC03 Port B DDR (data direction registers).

CIA #1: Complex Interface Adapter ($DC00-$DCFF)
-------------------------------------------------

Port Control:

$DC00   Port A                  Keyboard matrix and joystick #2
$DC01   Port B                  Keyboard matrix rows and joystick #1
$DC02   Port A DDR              Data direction register for Port A
$DC03   Port B DDR              Data direction register for Port B
