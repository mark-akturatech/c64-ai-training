# Commodore 64 Complete Memory Map - Source: sta.c64.org - SID Filter registers $D415-$D418: $D415 filter cutoff low (bits 0-2), $D416 filter cutoff high (bits 3-10), $D417 filter control (voice routing/resonance), $D418 master volume and filter mode register (write-only).

Filter ($D415-$D418):

$D415   Filter Cutoff L         Filter frequency bits 0-2 (write-only)
$D416   Filter Cutoff H         Filter frequency bits 3-10 (write-only)
$D417   Filter Control          Voice routing and resonance (write-only)
$D418   Volume/Filter           Master volume and filter modes (write-only)
