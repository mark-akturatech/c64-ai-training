# ********* - RS-232 'Receive the Next Bit' NMI subroutine: called by the NMI handler to input the next RS-232 data bit and then calls the next subroutine to reload the timer that causes subsequent interrupts.

65238         $FED6
RS-232 Receive the Next Bit (NMI)

The NMI handler calls this subroutine to input the next bit on the
RS-232 bus.  It then calls the next subroutine to reload the timer
that causes the interrupts.
