# Commodore 64 KERNAL Functions Reference - LSTNSA ($FF93). Transmits a LISTEN secondary address via the serial IEC bus (requires prior LISTEN call). Input: A = secondary address. Uses A register. Real address: $EDB9.

$FF93 - LSTNSA
Transmits LISTEN secondary address via serial bus (requires prior LISTEN call). Input: A = secondary address. Uses A register. Real address: $EDB9.
