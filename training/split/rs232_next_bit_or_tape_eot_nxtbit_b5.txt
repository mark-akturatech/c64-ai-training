# ********* - Documents NXTBIT ($B5), which holds the next bit to send for RS-232 output and is used by tape routines to indicate which part of a block the read routine is currently processing (effectively a tape EOT/position flag).

181           $B5            NXTBIT
RS-232 Next Bit to Send/Tape EOT Flag

This location is used by the RS-232 routines to hold the next bit to
be sent, and by the tape routines to indicate what part of a block the
read routine is currently reading.
